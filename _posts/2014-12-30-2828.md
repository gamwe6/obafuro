<p>CakePHP を動作させる際に phpMyAdmin を設定する際のことに関して質問です。</p>

<p>環境：</p>

<ul>
<li>CentOS6</li>
<li>PHP 5.5</li>
<li>CakePHP 2.6.0</li>
</ul>

<p>でやっています。</p>

<pre><code>/var/www/phpmyadmin/
</code></pre>

<p>に alias 指定でアクセスさせるために質問後半に示すような設定ファイルを書いた結果</p>

<pre><code>http://localhost/phpmyadmin/
</code></pre>

<p>でアクセスすることはできたんですが、</p>

<pre><code>File not Found
</code></pre>

<p>が出てしまいます…  <br>
読み込ませるにはどうすればいいでしょうか？</p>

<p>以下 <code>/etc/nginx/conf.d/default.conf</code> に記載した設定</p>

<pre><code>server {
    listen 80;
    server_name localhost;

    #中略
    location /phpmyadmin {
         alias /var/www/phpmyadmin;
         index index.php;

         location ~ \.php$ {
                  root /var/www/phpmyadmin;
                  fastcgi_pass 127.0.0.1:9000;
                  fastcgi_index index.php;
                  fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
                  include fastcgi_params;
         }
    }

    #中略

    #CakePHPのURLリライティング設定
    location ~ \.php$ {
         fastcgi_pass   127.0.0.1:9000;
         fastcgi_index  index.php;
         fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
         include        fastcgi_params;
    }
}
</code></pre>
