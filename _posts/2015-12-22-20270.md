以下のように記述してみましたが、変化はありません。</p>

<pre><code>    var http = require('http'),
  //サーバインスタンス作成
  server = http.createServer(function (req, res) {

      res.end('server connected');
  }),

<li>nginxの方でもAccess-Control-Allow-Originを書き換える指定を入れましたが、書き換えではなく追記となりました。</li>
