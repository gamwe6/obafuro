<p>Perl6 で配列のサイズが1しかないハッシュの配列を作りたいのですが、どういう訳か、ペア(Pair)の配列が作られてしまいます。</p>

<p>Perl6 ではこれが普通なのでしょうか、それとも私のプログラムの書き方が間違っているのでしょうか。ご指導の程よろしくお願いいたします。</p>

<pre><code>my @list = [
    { name =&gt; "foo", id =&gt; 1 }
];
@list.perl.say;
</code></pre>

<p>上の出力（ペアの配列）</p>

<pre><code>[:id(1), :name("foo")]
</code></pre>

<p>期待する出力（ハッシュの配列）</p>

<pre><code>[{:id(1), :name("foo")}]
</code></pre>
