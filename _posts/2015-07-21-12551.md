<p>これらの宣言って同じという認識で合ってますか？</p>

<p>配列</p>

```
var arr = [String]()
var arr = Array&lt;String&gt;()
```

<p>辞書</p>

```
var dic = [String: Int]()
var dic = Dictionary&lt;String, Int&gt;()
```
