<p>phpでexecを用いてpsqlを動かそうとしました。</p>

<pre><code>$cmd = '/usr/pgsql-9.3/bin/psql -U testuser example -c "SELECT * FROM test LIMIT 1"';
exec($cmd);
</code></pre>

<p>すると、</p>

<pre><code>$ php -f test.php
Password for user testuser:
</code></pre>

<p>とパスワード入力を求めるプロンプトが表示されてしまいます。</p>

<p>パスワードを渡す方法が知りたいです。
なお、サーバの設定は変更できません。</p>
