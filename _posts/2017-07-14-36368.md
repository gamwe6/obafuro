<p>たとえば、上記のモデルとデータで。</p>

<p>dog.nameが"taro" かつdog.ageが5以下のpersonsを得るには<br>
どのような検索を記述すればよいのでしょうか？</p>

<p>Dogを検索し、ownerをたどってpersonを得ることはできました。</p>

<pre><code>let dogs = realm.objects(Dog.self).filter(NSPredicate(format: "name = 'taro' &amp;&amp; age &lt;= 5"))
for dog in dogs {
    for p in dog.owners {
        NSLog(p.name)
    }
}
</code></pre>
