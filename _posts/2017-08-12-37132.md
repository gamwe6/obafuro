<p>Ruby の</p>

```
p (1..100).select{|i| i % 3 == 0 || i % 5 == 0}
```

<p>のようなプログラムをPARIで書きたいと思っています。<br>
すなわちある条件式を満たす要素だけ取り出すようなプログラムを書きたいのですがどうすればいいですか？<br>
P.S.<br>
select のオーバーフローする範囲にびっくりしたので追加の質問をします。<br>
簡単な質問で申し訳ないのですが、</p>

```
(1..10 ** 10).each{|i| p i if i % 3 == 0 || i % 5 == 0}
```

<p>のプログラムはPARIではどう書けばいいでしょうか？</p>
