<p>仕様1:2次元<br>
仕様2:線分と線分の交差は、気にしません。<br>
目的:輪(環)がいくつあるか知りたい。</p>

```
例1
input  : 線分1 [ 0, 0], [10, 0]
         線分2 [10, 0], [ 0,10]
         線分3 [ 0,10], [ 0, 0]
         線分4 [-5, 0], [-5,10]
output : 閉図形の数=1
       :  [0,0],[10,0],[ 0,10]

例2(8の字の形:1つとします)
input  : 線分1 [ 0, 0], [10,10]
         線分2 [10,10], [10, 0]
         線分3 [10, 0], [ 0,10]
         線分4 [ 0,10], [ 0, 0]
         線分5 [-5, 0], [-5,10]
output : 閉図形の数=1
       :  [0,0],[10,10],[10,0],[0,10]

例3(▽△)
input  : 線分1 [ 0, 0], [ 5, 5]
         線分2 [ 5, 5], [ 0,10]
         線分3 [10, 0], [ 0, 0]
         線分4 [ 5, 5], [10, 0]
         線分5 [10, 0], [10,10]
         線分6 [10,10], [ 5, 5]
output : 閉図形の数=2
       :  [0,0],[ 5, 5],[ 0,10]
       :  [5,5],[10, 0],[10,10]

例4(閉図形と閉図形の重なりはOKです)
```
