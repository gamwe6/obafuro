<p>以下のようなデータフレームdfがあるのですが、</p>

<pre><code>id category
1   3
2   2
3   3
4   1
5   2
6   2
</code></pre>

<p>これを</p>

<pre><code>id category1 category2 category3
1   0         0         1
2   0         1         0
3   0         0         1
4   1         0         0
5   0         1         0
6   0         1         0
</code></pre>

<p>のような形にしたいと考えています。<br>
現在はこれを</p>

<pre><code>df &lt;- tidyr::spread(df,category,category) #横展開
df.id &lt;- df[,1] #そのままやるとid列も1になってしまうため分離
df.other &lt;- df[,-1]
df.other[!is.na(df.other)] &lt;- 1
df.other[is.na(df.other)] &lt;- 0
df &lt;- cbind(df.id,df.other)
</code></pre>

<p>としているのですが、もう少しスマートな方法はないでしょうか？<br>
どなたかご教授お願い致します。</p>
