<p>Typescriptで以下のような定義をすると、</p>

<pre><code>class A {
  public b: A.B
}

namespace A {
  class B {
  }
}
</code></pre>

<p>classAの定義で</p>

<p><code>[ts] 名前空間 'A' にエクスポートされたメンバー 'B' がありません。</code></p>

<p>と怒られるし、定義順を逆にするとclassBの定義で</p>

<p><code>[ts] 名前空間宣言は、それとマージするクラスや関数より前に配置できません。</code></p>

<p>と怒られるのですが、いい解決方法はありますか？</p>
