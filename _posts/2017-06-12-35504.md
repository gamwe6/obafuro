<p>CSSでtranslate(10%)とすると、期待する移動距離になりません。（画像右端の青い隙間）<br>
translate(50px)とpxで指定するとうまくいきます。<br>
%の指定の際に留意する点などがあるのでしょうか？</p>

<p>[HTML]</p>

<pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="ja"&gt;
&lt;head&gt;
&lt;meta charset="utf-8"&gt;
&lt;title&gt;TEST&lt;/title&gt;

&lt;link href="styles.css" rel="stylesheet" media="all"&gt;
&lt;/head&gt;

&lt;body&gt;
    &lt;div class="container"&gt;
    &lt;div class="box box-left"&gt;

    &lt;/div&gt;
    &lt;div class="box box-right"&gt;

    &lt;/div&gt;
    &lt;/div&gt;
&lt;/body&gt;
&lt;/html&gt;
</code></pre>

<p>[CSS]</p>

<pre><code>*{
    margin:0;
    padding:0;
}

.container{
    width: 500px;
    background: blue;
    position:relative;
}


.box-left{
    width: 10%;
    height: 500px;
    background: tomato;
    position: absolute;
    left:0;
    top:0;
}


.box-right{
    width: 90%;
    height: 500px;
    background: green;
    transform: translate(10%);/*50pxとするとうまくいく*/
}
</code></pre>
