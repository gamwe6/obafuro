<p>スタック領域に積んだ値は、関数がネストしてもFrame Pointerから遡って参照することができると理解しているのですが、<br>
その場合、より深い場所にある値の上書きはできないのでしょうか?</p>

<p>また、できないとすればそれはなぜなのでしょう。<br>
セキュリティでしょうか?</p>

<p>ただし、代入する値のサイズがもとの値以下であり、Growableでないことを前提とします。</p>

<p>無駄に手書きですが、こういうイメージです</p>

```
 ーーーーーーー
|            |
|    関数     |
|            |
| ローカル変数 &lt;-|
 ----------- | |
|            | |  上書きしたい
| ネストした  ---
|   関数      |
|            |
 ーーーーーーー
```
