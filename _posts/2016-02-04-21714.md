<p><strong>現状</strong></p>

<pre><code>try {
  switch ($test) {
    case 1:
      break;
    case 2:
      throw new RuntimeException('');
    case 3:
      throw new RuntimeException('');
    default:
      throw new RuntimeException('');
  }
} catch (RuntimeException $e) {
}
</code></pre>

<hr>

<p><strong>やりたいこと</strong><br>
・2の時、try～catchを抜けたい<br>
・試してみたけど、うまくいかないコード</p>

<pre><code>try {
  switch () {
    case 1:
      break;
    case 2:
      $hoge = FALSE;
      break 2;
    case 3:
      throw new RuntimeException('');
    default:
      throw new RuntimeException('');
  }
} catch (RuntimeException $e) {
}
//ココへ飛ばしたい
</code></pre>

<p>　　</p>

<hr>

<p>もしくは、下記みたいなことは出来ないでしょうか？</p>

<pre><code>try {
  switch () {
    case 1:
      break;
    case 2:
      $hoge = FALSE;
      throw new HogeException('');
    case 3:
      throw new RuntimeException('');
    default:
      throw new RuntimeException('');
  }
} catch (RuntimeException $e) {
} catch (HogeException $e) {
  //ここで2だけを捕捉したい
}
</code></pre>
