<pre><code>ary1 = [[2, 5, 9], [5, 5, 6], [2, 6, 8], [2, 5, 7]]
ary2 = [[2, 9, 5], [5, 6, 5], [8, 6, 2], [7, 2, 5]]

# 最大値の小さい順
p ary1.sort{|a, b| a.max &lt;=&gt; b.max}
p ary2.sort{|a, b| a.max &lt;=&gt; b.max}
p ''
# 最大値の大きい順
p ary1.sort{|a, b| b.max &lt;=&gt; a.max}
p ary2.sort{|a, b| b.max &lt;=&gt; a.max}
p ''
# 最小値の小さい順
p ary1.sort{|a, b| a.min &lt;=&gt; b.min}
p ary2.sort{|a, b| a.min &lt;=&gt; b.min}
p ''
# 最小値の大きい順
p ary1.sort{|a, b| b.min &lt;=&gt; a.min}
p ary2.sort{|a, b| b.min &lt;=&gt; a.min}
</code></pre>

<p>出力結果<br>
[[5, 5, 6], [2, 5, 7], [2, 6, 8], [2, 5, 9]]<br>
[[5, 6, 5], [7, 2, 5], [8, 6, 2], [2, 9, 5]]<br>
""<br>
[[2, 5, 9], [2, 6, 8], [2, 5, 7], [5, 5, 6]]<br>
[[2, 9, 5], [8, 6, 2], [7, 2, 5], [5, 6, 5]]<br>
""<br>
[[2, 5, 9], [2, 5, 7], [2, 6, 8], [5, 5, 6]]<br>
[[2, 9, 5], [7, 2, 5], [8, 6, 2], [5, 6, 5]]<br>
""<br>
[[5, 5, 6], [2, 6, 8], [2, 5, 9], [2, 5, 7]]<br>
[[5, 6, 5], [8, 6, 2], [2, 9, 5], [7, 2, 5]]</p>

<p>四つ目が（たまたまかもしれませんが）[2, 6, 8], [2, 5, 9], [2, 5, 7]の順となっているのに、<br>
三つめが[2, 5, 9], [2, 5, 7], [2, 6, 8]の順となっています。</p>

<p>そこで質問なのですが、<br>
ブロックの部分を<br>
a.min &lt;=> b.min<br>
から<br>
b.min &lt;=> a.min<br>
と逆にしても得られる結果は<br>
ちょうど逆の並びにならない理由を教えてください。<br>
また、三つめの例で<br>
<strong>最小値が同じ場合は、次に小さいもの同士を比べ、小さい順に並べ<br>
さらに同じ場合は、その次に小さいもの同士を比べ、小さい順に並べる</strong><br>
ようにするにはどうすればいいですか？</p>

<p>（追記）<br>
三つめの例を<br>
<strong>最小値が同じ場合は、次に小さいもの同士を比べ、小さい順に並べ<br>

ようにする簡単な方法はあるようですが、<br>

<strong>最大値が同じ場合は、次に大きいもの同士を比べ、小さい順に並べ<br>

ようにする簡単な方法はあるのでしょうか？</p>
