```
sample = "アイス{}{}ー"
```

<p>という文があるとします。<br>
この中の <code>{}</code> を、分割をす‌​るパワーを持つもの、デ‌​ィヴィジョナー・分割者（独自用語です）と考えます。<br>
（デリミタというのは辞書で調べると、データの区切りにつけるマークということで、‌​たとえば、この場合だと、コンマ(<code>,</code>)だと‌​考えました。なので、分割をす‌​るパワーを持つものを、<code>{}</code>として、デ‌​ィヴィジョナーとしたつもりです）</p>

```
sample.split("{}")
```

<p>とすると、<br>
通常の結果は、</p>

```
["アイス","","ー"]
```

<p>となります。<br>
これを、</p>

```
["アイス","","","ー"]
```

<p>と出力する方法はありますか？</p>
