<p>Unityで新規プロジェクトを立ち上げ、いざスクリプトを作ろうとすると<br>
<code>Monobehaviourが見つからない。</code>となってしまいます。</p>

<p>以前はこんな事はなかったのですが、パソコンを変えバージョンなどを変えてからこの状態で<br>
一度アンインストールしてUnityをダウンロードしなおしても、ずっとこの状態のままです<br>
・・・素人の自分には何が原因で何を直せばいいのかがさっぱりわかりません</p>

<p><strong>実行環境</strong><br>
Mac OS X (10.14.4)<br>
Unity for mac ver 2019.1.3f1 personal<br>
visual studio ver 8.0.8(build2)</p>

<p><strong>エラーメッセージ</strong></p>

<pre><code>エラー：型または名前空間の名前”System"が見つかりませんでした。
  (usingディレクティブまたはアセンブリ参照が指定されている事を確認して下さい。)
   using ディレクトリは必要ありません。
</code></pre>

<p>こういったエンジンに関する項が全てエラーになっています。<br>
