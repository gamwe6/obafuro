<p>ウィンドウズのバッチファイルにおいて、 bash でいうところの -e オプション指定の動作を行うことは可能でしょうか。</p>

<p>具体的には、バッチ内で実行した各コマンドに対して exit code が 0 でないものがあった場合には、基本的にはその時点でバッチを abort させたい、というのが、やりたいことです。</p>

<p>もし、そんなものなどない、という場合には、上記のようなことがやりたい場合には、どのようにバッチファイルを記述するのでしょうか。各コマンドの後ろに %ERRORLEVEL% 判定を実行するしかないのでしょうか。</p>
