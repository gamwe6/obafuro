<p>実行例を次のようにするにはどうすればいいでしょうか?<br>
実行例：</p>

<pre><code>./a.out
1辺：2
**
*
</code></pre>

<p>ちなみに、自分は以下のようにプログラムしました。</p>

<pre><code>int main(void)
{
    int i,j,n;
    printf("一辺： ");
    scanf("%d",&amp;n);
    for(i=n;i&gt;0;i=i-1){
       for(j=n;i&lt;=j;j=j-1) {
        printf("*");
    }
    printf("\n");
    }
    return 0;
}
</code></pre>
