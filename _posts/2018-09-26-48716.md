<p>vscode extension api を利用し、動的にサイドバーのエクスプローラの背景色変更する方法が分かりません。(extension.ts上のロジックで色変更をしたい)</p>

<p>ちなみに、静的に変える方法は、下記手段を知っています。<br>
同様にプログラムで実現したいです</p>

<pre><code>"workbench.colorCustomizations": {
"list.activeSelectionBackground": "#a02e9b",
}
</code></pre>
