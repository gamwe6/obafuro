<!-- {% raw %} -->
<pre><code>.left{ flex: 0 100px; }
.right{ flex: 0 100px; }
.center { flex: 1; }
</code></pre>

<p><a href="https://i.stack.imgur.com/MkRdq.png" rel="nofollow noreferrer"><img src="https://i.stack.imgur.com/MkRdq.png" alt="画像の説明をここに入力"></a></p>

<p>を</p>

<pre><code>.left {
  flex: 100px;
  background: tomato;
}
.right {
  flex: 100px;
  background: pink;
}
.center {
  flex: 1;
  background: skyblue;
}
</code></pre>

<p>と書くと、<br>
下の画像のように、<br>
左右の100pxが効かなくなります。</p>

<p>省略されるのが、<strong>flex-grow</strong>になっているみたいです！</p>

<p><a href="https://i.stack.imgur.com/wxXe4.png" rel="nofollow noreferrer"><img src="https://i.stack.imgur.com/wxXe4.png" alt="画像の説明をここに入力"></a></p>

<p>デフォルトでは</p>

<p><strong>flex-grow:0<br>
flex-shrink:1<br>
flex-basis:auto</strong><br>
のはずですから、<br>
上の場合、</p>

<pre><code>.left{ flex: 0 1 100px; }
.right{ flex: 0 1 100px; }
.left{ flex: 1 1 auto; }
</code></pre>

<p>となっているかとおもいきや、<br>
これとは違う結果となります。<br>
どうも下のようになっているようです。</p>

<pre><code>.left {
  flex: 0 0 100px;
  background: tomato;
}
.right {
  flex: 0 0 100px;
  background: pink;
}
.center {
  flex: 1 0 auto;
  background: skyblue;
}
</code></pre>

<p>3つとも<br>
<strong>flex-shrinkが0</strong>になっているようです！</p>

<pre><code>.left{ flex: 1 100px; }
</code></pre>

<p>と書いた時、<br>
省略されるのはどのプロパティですか？</p>
