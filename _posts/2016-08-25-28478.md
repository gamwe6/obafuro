<p>Swiftを参考書を見てプログラミングしていく中で、<code>func ~</code>で定義した関数で</p>

<p>普通は</p>

```
func touch(){
    ...
}

touch()
```

<p>のように扱うことは分かりました。</p>

<p>だけれど、</p>

```
func touch(){
    ...
}
```

<p>だけで実行できる関数もあります。<br>
どうして、実行できるのでしょうか？</p>

<p>実行できる関数には、引数に<code>UI〜</code>を持つものが多い気がするのですが何か関係しているのでしょうか？</p>
