<p>例えば以下のクラスCでsayメソッドを使ったら<br>
インスタンス変数@dはnilで初期化されています．</p>

<p>このメソッドは３行にわたって書いており，<br>
もうすこしスマートに書けないかと悩んでいます．</p>

<p>すでに機能的には充足されていますが，<br>
rubyらしく書くためにはどう工夫すればよいでしょうか</p>

<pre><code>class C
  def initialize
    @d = "hello"
  end
  def say
    result = @d
    @d = nil
    result
  end
end

c = C.new
c.say #=&gt; hello
c.say #=&gt; nil
</code></pre>
