<pre><code>&lt;input type="text" class="input"&gt;
&lt;input type="text" class="input"&gt;　　
&lt;input type="text" class="input"&gt;
&lt;input type="text" class="input"&gt;
&lt;input type="text" class="input"&gt;
</code></pre>

<p>上記のようなHTMLにて、<br>
入力されているinputタグに処理をしたいのですが、</p>

<pre><code>$('.input').each(function() {
  if ($(this).val()) {
    $(this).hogehoge;
  }
});
</code></pre>

<p>↑のようにループ等使わずに、</p>

<pre><code>$('xxxxxx').hogehoge;
</code></pre>

<p>と、1行で書く事はできないでしょうか。<br>
