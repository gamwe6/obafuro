<pre><code>int a = -2147483648;
int b = a * -1; // -2147483648
</code></pre>

<p>32ビットの signed int の値の範囲が</p>

<blockquote>
  <p>-2,147,483,648 ～ 2,147,483,647</p>
</blockquote>

<p>であることから、b の値が +2147483648 になり得ないことは分かります。<br>
ただ、-2147483648 になる理由が分かりません。<br>
C# だけでなく Java などでも同様のようです。<br>
仕様と言ってしまえば、それまでなのでしょうが、何か合理的な理由があるのでしょうか？</p>

<pre><code>a = (a * -1) * -1
</code></pre>

<p>という式が成り立つようにするためでしょうか。</p>
