<p>Rの関数化についての質問です。</p>

<p>関数の中身のfor文だけ実行すると破壊的になりうまくいきますが、、関数として実行すると非破壊的になって更新されていないようです。<br>
グローバル変数が原因かと想像しているのですが、&lt;&lt;-を使ってもうまく行かず、という感じです。</p>

```
func &lt;- function(x1,df){
  for(i in 1:nrow(df)){
    df[i,x1+1] &lt;- min(df[i,1:x1]) 
     }
}
```
