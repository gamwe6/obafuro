<p>Pythonのpropertyについて質問です。<br>
よく見かけるコードでは以下のようにインスタンス変数の前にアンダースコアがついていますが、<br>
これはどういう意味なのでしょうか？<br>
propertyを使うときには必要なのですか？</p>

<pre><code>class MyClass:

    def __init__(self, x):
        self._x = x

    @property
    def x(self):
        return self._x

    @x.setter
    def x(self, x):
        self._x = x
        do_something()
</code></pre>
