<p>Swiftではmap,filterなどの便利な機能があるのであまりfor文を使わない方がいいと聞いたのですが、以下のような場合も書き換え可能な方法がありますでしょうか。</p>

<pre><code>var test = ["りんご","ごりら","らっぱ","ぱらしゅーと","とんねる"]
for i in 0..&lt;test.count{
    if test[i] == "ごりら"{
        test[i] = "ごじら"
    }
}
</code></pre>

<p>要素をすべて確認して、一致する場合、一致した要素そのものに変更を加えたいのです。</p>
