<p>質問:</p>

<ul>
<li>ubuntu を USB にインストールしてそこからブートさせることはできますか？</li>
</ul>

<p>環境:</p>

<ul>
<li>Ubuntu 16.04</li>
<li>Mac OS X El Capitan</li>
</ul>

<p>やったこと:</p>

<p>MacBookAir と USB*2 を用意して、次を実行しました。</p>

<ol>
<li>公式のインストーラー ISO を USB に焼く</li>
<li>インストーラー USB をブートし、もう一つの USB (以下、対象USB) に向かって ubuntu インストール実行</li>
<li>インストールが終わった段階で、インストーラー USB を抜いて、対象 USB のみがささった状態でブート対象を選択する方法で PC 再起動 (起動時に <kbd>alt/option</kbd>長押し) しても、 ubuntu の USB は表示されません。</li>
</ol>
