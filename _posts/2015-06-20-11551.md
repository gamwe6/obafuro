<p>i (1以上9以下)を含むならば、i がi 個含む数について、<br>
オンライン整数列大辞典のA108571（<a href="https://oeis.org/A108571" rel="nofollow">https://oeis.org/A108571</a>）に載っています。<br>
また、この条件をみたす数の個数は <br>
66712890763701234740813164553708284 <br>
であることも載っています。<br>
この個数を次のコードで求めました。</p>

<pre><code>def factorial(m)
  (1..m).inject(:*)
end

def c(ary)
  sum = 0
  p = 1
  ary.each{|m|
    sum += m
    p *= factorial(m)
  }
  factorial(sum) / p
end

n = 10
s = 0
(1..n - 1).each{|i|
  # i個の文字を使用
  (1..n - 1).to_a.combination(i){|ary|
    s += c(ary)
  }
}
p [n, s]
</code></pre>
