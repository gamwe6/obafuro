<pre><code>brew install make
</code></pre>

<p>にてGNU makeをインストールしました。</p>

<pre><code>which make
/usr/local/bin/make
</code></pre>

<p>であり、<br>
    /usr/local/bin/make --version<br>
とすると<br>
GNU Make 4.2.1 <br>
となりますが、</p>

<pre><code>make --version
</code></pre>

<p>とすると<br>
GNU Make 3.81 <br>
となってしまいます。</p>

<pre><code>echo $PATH
/usr/local/bin:/usr/bin:/bin:/usr/sbin:/sbin:
</code></pre>

<p>なので、パスは通っているはずです。</p>

<p>mac付属のコマンドより優先度を上げるにはどうすれば良いのでしょうか？</p>
