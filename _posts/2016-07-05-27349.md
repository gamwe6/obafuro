<p>git の tree-ish には、いくつかの書式があります。</p>

<pre><code>develop
develop^
HEAD
develop:some-dir
</code></pre>

<p>これらを引数に、 tree object の id そのものを取得するにはどうしたらいいですか？</p>

<p>背景: <code>git commit-tree</code> が tree-ish ではなく、 tree object の id でないと動いてくれなかったので。</p>
