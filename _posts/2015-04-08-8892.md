<pre><code>hoge.addEventListener( 'click', function() {
    return function() {
  }
}());
</code></pre>

<p>下記は、イベントハンドラ？　コールバック関数？　もしくは両方？</p>

<pre><code>function() {
    return function() {
  }
}()
</code></pre>

<p>それとも下記が、イベントハンドラ？　コールバック関数？　もしくは両方</p>

<pre><code>return function() {
}
</code></pre>

<p>あるいはコールバック関数は、外部で呼出形式のものを指すのでしょうか？<br>
