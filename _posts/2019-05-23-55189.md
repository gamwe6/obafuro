<p>下記のようなpandas.DataFrameがある時に、縦持ちに変換を行いたいです。<br>

<pre><code>import pandas as pd
df1 = pd.DataFrame([[0, 'e1', 'e11'], [1, 'e1', 'e22'], [2, 'e1', 'e33']], columns=['id', 'c1', 'c2'])
df1

# id c1 c2
# 0 e1 e11
# 1 e1 e22
# 2 e1 e33
</code></pre>

<p>変換後</p>

<pre><code># id c_name value
# 0 c1 e1
# 0 c2 e11
# 1 c1 e1
