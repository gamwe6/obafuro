<p>自分のプログラムのどこが間違っていますか。</p>

<pre><code>int sum(int n)
{
    int s,i;
    s=0;
    for(i=0;i&lt;=n;i=i+1) {
        s=s+i;
    }

    return s;
}

int main(void)
{
    int n,ans;
    printf("n: ");
    scanf("%d",&amp;n);
    ans=sum(n);
    printf("sum(n)=%d \n",ans);

    return 0;
}
</code></pre>
