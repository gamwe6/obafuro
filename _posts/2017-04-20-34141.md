<p>Rust の質問です. 以下のコードはコンパイルエラーになります.</p>

<pre><code>fn main() {
    let a = 100;
    let b = [0; a];
}
</code></pre>

<p>エラー:</p>

<pre><code>let b = [0; a];
            ^ unresolved path in constant expression
</code></pre>

<p><code>let a</code> してるのに, どうして <code>a</code> が解決されていないとされてしまってるのでしょうか. よろしくお願いします.</p>
