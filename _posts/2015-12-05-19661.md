<p>大きな自然数nに対しても、<br>
フィボナッチ数列f(n)の下20桁を高速に求める<br>
にはどうすればよろしいでしょうか？</p>

<p>なお、私は以下のように<br>
f(n)を高速に求め（※）、mod をとりました。</p>

<pre><code># (a + b√x)^nの計算
def power(a, b, x, n)
  return [1, 0] if n == 0
  c, d = power(a, b, x, n &gt;&gt; 1)
  c, d =
  c * c + x * d * d,
  2 * c * d
  return [c, d] if n &amp; 1 == 0
  c, d =
  a * c + x * b * d,
  a * d + b * c
  return [c, d]
end

# (1 + √5)^n = an + bn√5とすると、f(n) = bn / 2^(n - 1)
def f(n)
  power(1, 1, 5, n)[1] &gt;&gt; n - 1
end

Mod = 10 ** 20
(0..8).each{|i| p f(9 ** i) % Mod}
</code></pre>

<p>※次のように「f(n)を行列の累乗計算を使って求める方法」があるが、<br>
上記の方が少し速い。</p>

<pre><code>require 'matrix'

def f(n)
  v = Vector[1, 0]
  a = Matrix[[1, 1], [1, 0]]
  ((a ** n) * v)[1]
end
