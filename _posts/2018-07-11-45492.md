<p>現在、 Python 環境上で次のようにして Axes3D を利用しています。</p>

```
from mpl_toolkits.mplot3d import Axes3D
import matplotlib.pyplot as plt
```

<p>ところが、この Axes3D は座標軸の座標目盛間隔を自動的に決めてしまうので、表示されたものの形が変わってしまいます。</p>

<p>そこで、 Axes3D は座標軸の座標目盛間隔を等間隔にしたいと思っています。<br>
どうすればよろしいでしょうか。</p>
