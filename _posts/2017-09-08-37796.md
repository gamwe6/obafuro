<p>引数 n が負の整数でなく、<code>アルゴリズムの実行時間 = 代入の数 + 数を加算する数</code>とする場合のビッグオー、ビッグシータの求め方について、</p>

<p>(1)</p>

<pre><code>fnc(n){
    a=0 // +1回
    if nが奇数: 
         for i = 1 to n: // +n回
            a += 1;      // +n回
    return a;
}
</code></pre>

<p>↑の関数の場合nが奇数なら合計1+n*2 = 2n+1 の実行時間となるのでビッグオーで表すとO(n)ということでよいでしょうか？</p>

<p>(2)</p>

<pre><code>fnc(n){
    if n &lt; 10:
        return 0;
    return fnc(n - 3) + 1;
}
</code></pre>

<p>(2)に関してはビッグオーの求め方自体がわかりません。再帰処理がある場合のビッグオーはどのようにして求めたら良いのでしょうか？</p>
