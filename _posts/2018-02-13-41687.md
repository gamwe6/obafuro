<p>どうすればうまくいくでしょうか？</p>

<p>特にわからなかったとこがIAMの設定で，<br>
・ロールポリシーの作成 をクリック<br>
・Policy Generator の枠内にある 選択 をクリック<br>
とうがどこを押せばいいのかわかりませんでしたので以下のように設定しました。</p>

<pre><code>{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "VisualEditor0",
            "Effect": "Allow",
            "Action": [
                "mobileanalytics:PutEvents",
                "cognito-sync:*"
            ],
            "Resource": "*"
        },
        {
            "Sid": "VisualEditor1",
            "Effect": "Allow",
            "Action": "s3:*",
            "Resource": "*"
        }
    ]
}
