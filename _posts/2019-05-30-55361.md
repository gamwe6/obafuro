<p>Golangで構造体を初期化するとき、No.1とNo.2では同じ出力が得られると思うのですが、どう使い分けたらいいのでしょうか？</p>

<pre><code>package main

import (
    "fmt"
)

type Test struct{
    A string
}

func main() {
    test1 := new(Test) // No.1
    test2 := &amp;Test{}   // No.2

    fmt.Println(test1)
    fmt.Println(test2)
}


</code></pre>
