<p>下記関数のSwitch文について、質問させて下さい。</p>

<pre><code>func f1(obj: AnyObject) -&gt; AnyObject {
    switch obj {
    case let arr as NSArray:
        return arr
    case let dic as NSDictionary:
        return dic
    default:
        return obj
    }
}
</code></pre>

<p>引数のobjに配列を入れれば、</p>

<pre><code>case let arr as NSArray:
    return arr
</code></pre>

<p>が実行され、<br>
連想配列を入れれば、</p>

<pre><code>case let dic as NSDictionary:
    return dic
</code></pre>

<p>が実行されます。</p>

<p>objが配列、連想配列のいずかにcastできればtrueが返ってきて、分岐されるのだと思うのですが、<br>
「case let arr as NSArray:」の表記方法は何かの略なのでしょうか？<br>
Optional Bindingの一種？</p>
