<!-- {% raw %} -->
<p>通常のクラスでは、自身のクラス名は、問題なく解決できます。</p>

<pre><code>class Hoge
  def new_my_class
    Hoge.new
  end
end

h = Hoge.new
p h.new_my_class
# =&gt; #&lt;Hoge:0x000000032f6b58&gt;
</code></pre>

<p>BasicObject を継承したクラスでは、これができないことがわかりました。</p>

<pre><code>class HogeBasic &lt; BasicObject
  def new_my_class
    HogeBasic.new
  end
end

h = HogeBasic.new
p  h.new_my_class
# # 以下の例外が発生する。
# NameError: uninitialized constant HogeBasic::HogeBasic
# Did you mean?  HogeBasic
</code></pre>

<p>これは、なぜなのでしょうか？</p>
