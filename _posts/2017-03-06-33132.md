<p>上記のようなオブジェクトから指定のキーを検索して、その値を設定したいと考えています。</p>

<pre><code>function searchObj (obj, query) {
  for (var key in obj) {
    var value = obj[key]
    if (typeof value === 'object') {
      searchObj(value, query)
    }
    if (key === query) {
      return obj[key]
    }
  }
}
</code></pre>

<p>関数を設定して、返り値でもらったプロパティキーの値を設定するようにしました。</p>

<pre><code>var key = searchObj(state.item[0], 'x'))

</code></pre>
