<p>c++の仮想関数について質問です。<br>
次のように非常に単純なBaseとそれを継承したDerivedを用意します。</p>

<pre><code>#include &lt;iostream&gt;

struct Base {
virtual void f()
{
   std::cout &lt;&lt; "Base" &lt;&lt; std::endl;
}
};

struct Derived : Base {
void f() override
{
    std::cout &lt;&lt; "Derived" &lt;&lt; std::endl;
}
};

int main()
{
  {
    Derived d;
    Base b = d;

    b.f();
  }

  {
    Derived d;
    Base&amp; b = d;

    b.f();
  }
}
</code></pre>

<p>実行結果が、</p>

<pre><code>Base
Derived
</code></pre>

<p>となります。</p>

<p>なぜ前者ではBaseのf()が呼ばれてしまうのでしょうか。</p>
