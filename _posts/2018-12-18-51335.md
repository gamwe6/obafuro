<p>Juliaにおいて、２次元配列の最後尾の指定の仕方はあるのでしょうか？（自分が調べた限りではなさそうです）</p>

```
array = zeros(Int64,3,3)
array[2,:]=[1,1,1]
array[3,:]=[2,2,2]
```

<p>の２次元配列があるとして、最後尾をコピーしたい場合</p>

```
a = copy(array[3,:])
```

<p>ではなく、もう少し一般化して書きたいということです。<br>
例えば配列指定にlength(array)を使うなどです。<br>
