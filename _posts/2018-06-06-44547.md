<p>以前作ったクラスに新しくメソッドを追加したいのですが上手くいきません。<br>
目的は新しくdraw_barというメソッドをつかってグラフを書きたいと考えています。</p>

<p>［以前作ったもの］</p>

<pre><code>import turtle

class Kame(turtle.Turtle):
    def __init__(self):
        super().__init__()
        self.shape("turtle")
        self.shapesize(2,2)
</code></pre>

<p>［追加したいメソッド］</p>

<pre><code>def draw_bar(self,height,width=40):
    self.left(90)
    self.fd(height)
    self.right(90)
    self.fd(width)
    self.right(90)
    self.fd(height)
    self.left(90)
</code></pre>

<p>［ターミナルでの実行コード］</p>

<pre><code>import kame
&gt;&gt;&gt; hist_kame=kame.Kame()
&gt;&gt;&gt; hist_kame.draw_bar(120)
</code></pre>

<p>［エラー］<br>
AttributeError: 'Kame' object has no attribute 'draw_bar'</p>
