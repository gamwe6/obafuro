<p>[環境]<br>
Ruby 2.3.0<br>
Windows 7 32bit 日本語環境</p>

<p>上記の環境において、中国語が含まれるファイルをリネームしようとするとエラーが発生します。<br>
正常に処理するためには、どうすればよいのかアドバイスお願いします。<br>
下記がサンプルプログラムです。</p>

<pre><code>directory = '/hoge/'
count = 1
Dir::entries(directory).each{|d|
    File::rename( directory + d , count )
    count = count + 1
}
</code></pre>
