<p>次のように，<code>Vec&lt;T&gt;::iter()</code>をイテレータを消費する関数<code>consume()</code>に対して渡したいのですが，</p>

<pre><code>fn consume&lt;T: Iterator&lt;Item=char&gt;&gt;(input: &amp;mut T){
    unimplemented!()
}

fn main(){
    let mut buffer = vec!['a','b','c'];
    consume(&amp;mut buffer.iter()); //type mismatch
} 
</code></pre>

<p>次のようなコンパイルエラーが生じます．</p>

<pre><code>prog.rs:7:5: 7:12 error: type mismatch resolving `&lt;core::slice::Iter&lt;'_, char&gt; as core::iter::Iterator&gt;::Item == char`:
  expected &amp;-ptr,
found char [E0271]
</code></pre>

<p><code>buffer</code>は<code>consume()</code>の呼出し後は破壊されても構いません．どのように書くのが良いでしょうか．</p>
