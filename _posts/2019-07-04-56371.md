<h1>環境</h1>

<ul>
<li>Python 3.6</li>
</ul>

<h1>背景</h1>

<p>PythonでCLIツールを作成しています。<br>
あるWebサービスから情報を取得して、以下のようなフォーマットでCSVファイルを出力するようなCLIです。<br>
実際には20～40列あります。</p>

<pre><code>name,a_seconds,b_seconds
Alice,3600,1800
Bob,3600,3600
</code></pre>

<p>コマンドは以下のように実行します。</p>

<pre><code>$ python print_csv.py 
</code></pre>

<h1>やりたいこと</h1>

<p>CSVのいくつかの列は、ユーザがカスタマイズして出力できるようにしたいです。<br>
たとえば、<code>a_b_hours</code>列を出力したい場合、以下のような設定をコマンドの外から渡せるようにしたいです。</p>

<pre><code>column_name = "a_b_hours"

def column_func(df):
    df[column_name] = df["a_secons"] + df["b_seconds"]
</code></pre>

<p>出力結果</p>

<pre><code>name,a_seconds,b_seconds,a_b_hours
Alice,3600,1800,1.5
Bob,3600,3600,2
</code></pre>

<pre><code>$ python main.py 3
num = 3

$ python main.py 3 --script custom.py

</code></pre>
