<p>下記、3つの配列を生成します。</p>

```
r = []
3.downto(1) do |x|
  k = gets.split(' ')　　　#数字・半角スペース・数字　と入力
  k = k.map { |v| v.to_i}
  r &lt;&lt; k
end
p r　　　　　　　　        　#=&gt;　[[1, 2], [3, 4], [5, 6]]
```

<p>要素の操作例</p>

```
p r[0][1] ** 2　　　     　#=&gt;　4
```

<p>▼本当は <code>m = [1, 2]</code>, <code>n = [3, 4]</code>, <code>o = [5, 6]</code> とできれば要素の操作がわかりやすいと思っていますが、やり方・可不可がわかりません。<br>
　ご教授いただけますと幸いです。よろしくお願いいたします。</p>

<p>【追記】<br>
　例えば、複数人の買い物金額を、雑貨、食料品、書籍の順に<br>
m:1000 1100 1200　　n:1300 1400 1500　　o:1600 1700 1800<br>
　と入力した後で、(これも例ですが、1100に2日分とか3日分ということで2や3〜を掛けたり掛けなかったりして)各人の合計を出す、というものです。<br>
　r = [[1000, 1100, 1200], [1300, 1400 1500], [1600, 1700, 1800]]<br>
よりも<br>
　m = [1000, 1100, 1200]　n = [1300, 1400, 1500]　o = [1600, 1700, 1800]<br>
の方がわかりやすい（r[0][1] と m[1]）し可読性も増すと思われます。</p>
