<p>■ルーティングに関する質問です。</p>

<p>とある事情があり、ネットワークに負荷のかかる動画閲覧のみ、<br>
USBで外付けたLANアダプタ経由で行いたいです。</p>

<p>USBで外付けたLANアダプタの次(ネクストホップ)は自宅のルータです。<br>
ルータ上でプロバイダ上にPPP接続しています。</p>

<p>ここで以下のようなルーティングが考えられます。<br>
route -p add XXX.XXX.XXX.XXX mask 255.255.255.255 192.168.11.1 metric 1 if 37</p>

<p>しかしyoutubeなど大手動画サイトは負荷分散を行っておりIPが動画閲覧の都度(閲覧中にも)変更となるので、<br>
全てのIPを指定するのは不可能です。</p>

<p>どのようにすれば固定の動画サイトのルーティングを行えるのでしょうか？</p>

<p>イメージとしては以下のようにしたいのです。<br>
route -p add www.youtube.com mask 255.255.255.255 192.168.11.1 metric 1 if 37</p>
