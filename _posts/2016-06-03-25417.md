```
$a1 $a2 $a3 
$b1 $b2 $b3 
$c1 $c2 $c3
$d1 $d2 $d3
```

<p>このような変数があったとき、変数[1-3]全てにおいてnullだった場合に上に詰めたいです。<br>
フォームからとるので、どこがnullになるかわかりません。</p>

<p>アナログに</p>

```
if($a1==""&amp;&amp;$a2==""&amp;&amp;$a3==""){
$a1=$b1
}
```

<p>などとしようとしましたがD列について書こうとするとかなり大変で。<br>
すっきりと詰めることができればと思い質問します。</p>
