<p>コード:</p>

<pre><code>struct hoge {
    var a: Int = 0
}

var obj: hoge = hoge(a: 10)
println(obj.a) //一度目のprintln
obj.a = 5
println(obj.a) //二度目のprintln
</code></pre>

<p>出力結果:</p>

<pre><code>10
5
</code></pre>

<p>コード:</p>

<pre><code>struct hoge {
    var a: Int = 0
}

var obj: hoge = hoge(a: 10)
//println(obj.a)　//一度目のprintlnをコメントアウト
obj.a = 5
println(obj.a) //二度目のprintln
</code></pre>

<p>出力結果:</p>

<pre><code>10
</code></pre>

<p>上記の様に、一度目のprintlnを実行するかしないかで、二度目のprintlnの結果が変わってきます。（「obj.a = 5」が処理されていない？）</p>
