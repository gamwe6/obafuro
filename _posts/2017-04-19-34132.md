```
var text = '11,222,3,4444';
```

<p>たとえば、この文字列の3回目に出現するカンマを*に置換したい場合どのようにすればよいでしょうか？</p>

```
// =&gt; '11,222,3*4444';
```

<p>カンマに限らず【】で囲まれた場合なども対応できるようにしたいです。</p>

```
// 2番目に出現する【】で囲まれた文字列を**に置換したい
var text = 'あ【あ】いい【い】うえお';
// =&gt; 'あ【あ】いい**うえお'
```
