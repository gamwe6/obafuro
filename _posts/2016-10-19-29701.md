<p>MacでHomebrewを使ってOpenCV3をインストールし、<br>
下記「sample.cpp」というファイルを作成しました。</p>

<p>【sample.cpp】</p>

<pre><code>    #include &lt;opencv2/opencv.hpp&gt;
int main()
{
  cv::VideoCapture capture(0);
  for(int i=0;i&lt;5;i++)
  {
    cv::Mat readImg;
    capture &gt;&gt; readImg;
    cv::imshow("Captured Image",readImg);
    cv::waitKey(0);
  }
  return 0;
}
</code></pre>

<p>ターミナルから下記文言でコンパイルを試みましたが</p>

<pre><code>$ clang++ -o sample sample.cpp
</code></pre>

<p>下記エラーが表示されました。</p>

<p>こちらのエラーの原因がわかりません。</p>

<p>また、教本を見ながら下記でコンパイルを試みました。</p>

<pre><code>$ clang++ -o sample 'pkg-config --cflags opencv' 'pkg-config --libs opencv' sample.cpp
</code></pre>

<p>こちらは下記エラーが出力されました。</p>

<pre><code>clang: error: no such file or directory: 'pkg-config --cflags opencv'
clang: error: no such file or directory: 'pkg-config --libs opencv'
</code></pre>

<p>ファイルがないのかなと思いましたが下記文言を打ち込むと</p>

<pre><code>$ pkg-config --cflags opencv
</code></pre>

<p>下記のように出力されます。</p>

<pre><code>-I/usr/local/Cellar/opencv/2.4.13_3/include/opencv -I/usr/local/Cellar/opencv/2.4.13_3/include
</code></pre>

<p>下記文言は</p>

<pre><code>$ pkg-config --libs opencv
</code></pre>

<p>下記のように出力されます。</p>

<pre><code> -L/usr/local/Cellar/opencv/2.4.13_3/lib -lopencv_calib3d -lopencv_contrib -lopencv_core -lopencv_features2d -lopencv_flann -lopencv_gpu -lopencv_highgui -lopencv_imgproc -lopencv_legacy -lopencv_ml -lopencv_nonfree -lopencv_objdetect -lopencv_ocl -lopencv_photo -lopencv_stitching -lopencv_superres -lopencv_ts -lopencv_video -lopencv_videostab
</code></pre>

<p>いつもc/c++はコンパイルできます。<br>
なぜかopencvだけうまくコンパイルできません。</p>
