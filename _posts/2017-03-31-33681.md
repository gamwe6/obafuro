```
num_list = [[0 * 100]] * 2 
for i in range(4): 
    for j in range(100): 
        num_list[i].append(j)

print(num_list)
```

<p>として</p>

```
[[0, 1, 2, 3 -- 100], [0, 1, 2, 3 -- 100]]
```

<p>のような配列を作りたいのですが、結果として </p>

```
[[0, 1, 2, 3 -- 100, 0, 1, 2, 3 -- 100],[0, 1, 2, 3 -- 100, 0, 1, 2, 3 -- 100]]
```

<p>となってしまいます。</p>

<p>明らかにイテレーションの理解不足だと思うのですが、 <br>
どなたかご助言していただけないでしょうか。</p>
