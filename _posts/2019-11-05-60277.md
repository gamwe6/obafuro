<p>AWS の cli を叩いて、リソースがない場合に作成みたいなことを Makefile の中でやりたいです</p>

<p>シェル変数でも Makefile の変数でもいいのですが<br>
以下のようなテストコマンドをいろいろかいてみたんですがどうやっても空文字にマッチするIF文に入ってくれません</p>

<ol>
<li>シェル変数を使う場合</li>
</ol>

<pre><code>test1:
    export A=1
    echo $A
</code></pre>

<p>エクスポートしても別の行でとり出せない</p>

<pre><code>#make test1
export A=1
echo
</code></pre>

<ol start="2">
<li>eval のテスト</li>
</ol>

<pre><code>test2:
    $(eval A := 1)
    echo $(A)
ifeq (A, "")
    $(eval A := 2)
endif
    echo $(A)
</code></pre>

<p>これは正常</p>

<pre><code>#make test2
echo 1
1
echo 1
1
</code></pre>

<p>3〜5. 実際に判定してほしいコマンド （違いは ifeq 文のみです)</p>

<pre><code>test3:
    $(eval A := $(shell aws apigateway get-rest-apis | jq -r '.items[] | select(.name=="test") | .id')))
    echo $(A)
ifeq (A, "")
    $(eval A := 2)
endif
    echo $(A)

test4:
    $(eval A := $(shell aws apigateway get-rest-apis | jq -r '.items[] | select(.name=="test") | .id')))
    echo $(A)
ifeq ("A", "")
    $(eval A := 2)
endif
    echo $(A)

test5:
    $(eval A := $(shell aws apigateway get-rest-apis | jq -r '.items[] | select(.name=="test") | .id')))
    echo $(A)
ifeq ($(A), "")
    $(eval A := 2)
endif
    echo $(A)
</code></pre>

<p>どれも api-gateway が定義されてない場合空文字になって<br>
最後に 2 になってほしいのにならない</p>

<pre><code>#make test3
echo

echo

#make test4
echo

echo

#make test5
echo

echo

</code></pre>

<p>どうすればコマンド結果が空かどうかを判定できるでしょうか</p>

<p>もっと根本的に Makefile 内で<br>
aws cli の結果によってリソースを作成するという簡単な方法があったりするでしょうか</p>

<p>make のバージョンは以下です</p>

<pre><code># make -v

</code></pre>

<h3>追記：</h3>

<p>コメントで教えていただいたリンク先を参照して</p>

<pre><code>test6:
    $(eval A := $(shell aws apigateway get-rest-apis | jq -r '.items[] | select(.name=="test") | .id'))
    echo $(A)
ifeq ($(A),)
    $(eval A := 2)
endif
    echo $(A)
</code></pre>

<p>という書き方を試したところ</p>

<pre><code># make test6
echo

echo 2
2
</code></pre>

<p>と IF に入ってくれたのですが</p>

<p>逆に空文字以外でも必ず IF 文にマッチしてしまいます</p>

<pre><code>test7:
    $(eval A := 1)
    echo $(A)
ifeq ($(A),)
    $(eval A := 2)
endif
    echo $(A)
</code></pre>

<pre><code>#make test7
echo 1
1
echo 2
2
</code></pre>

<p>どこか書き方がまずいのでしょうか</p>
