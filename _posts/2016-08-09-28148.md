<p>データベース用にしていたPCが壊れてしまい、/var/lib/mysql を丸ごと別環境に移設しました。<br>
この時、生きているテーブルと、死んでいるテーブルがあり、死んでいる方は、次のエラーが出ていました。</p>

<pre><code>mysql&gt; select * from wp_test_users;
ERROR 1273 (HY000): Unknown collation '#224' in table 'wp_test_users' definition
</code></pre>

<p>調べてみると、Collation 参照順序 のエラーであることは理解しました。</p>

<pre><code>ALTER TABLE "Database" CONVERT TO CHARACTER SET utf8 COLLATE utf8_general_ci
</code></pre>

<p>等のコマンドも試しましたが、このコマンド自体も上記と同じエラーとなり、どうすれば良いか分かりません。<br>
ご教授願えませんでしょうか。よろしくお願いします。</p>
