<p>linuxのsedコマンドでtest.txtの3番目に出現するLinuxをリナックスに置換したいです。</p>

<pre><code>test.txt
    Linux
    Linux
    Linux
    Linux
    Linux
</code></pre>

<p><strong>私が実行したコマンド</strong><br>
<code>sed -i -e 's/Linux/リナックス/3' test.txt</code></p>

<p>しかし実際には全ての行のLinuxがリナックスに置換されてしまいます。</p>

<pre><code>test.txt
    リナックス
    リナックス
    リナックス
    リナックス
    リナックス
</code></pre>

<p>どなたか、アドバイスして頂けたら幸いです。</p>

<p><strong>捕捉</strong><br>
置換したい対象は行方向に見ていった時、何行目にあるいか分からないが、<br>
n番目(この場合は3番目)に出現するLinuxを置換したいというものです。</p>

<p><strong>置換対象のパターン</strong><br>
パターン1<br>
入力</p>

<pre><code>test.txt
    android
    Linux
    windows
    windows
    Linux
    macos
    Linux
    android
    Linux
</code></pre>

<p>出力</p>

<pre><code>test.txt
    android
    Linux
    windows
    windows
    Linux
    macos
    リナックス&lt;&lt;3番目に出現したので置換
    android
    Linux
</code></pre>

<p>パターン2<br>
入力</p>

<pre><code>test.txt
    android
    Linux
    windows
    windows
    LinuxLinux
    macos
    Linux
    android
    Linux
</code></pre>

<p>出力</p>

<pre><code>test.txt
    android
    Linux
    windows
    windows
    Linuxリナックス&lt;&lt;3番目に出現したので置換
    macos
    Linux
    android
    Linux
</code></pre>
