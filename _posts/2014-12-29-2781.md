<p>JavaScriptでの関数定義には2つあるみたいですが、違いや使い分けがわからないので教えて下さい。</p>

<p>パターン1</p>

<pre><code>function hoge(a, b) {
    ...
}
</code></pre>

<p>パターン2</p>

<pre><code>var hoge = function(a, b) {
    ...
}
</code></pre>

<p>どちらも実行するときは<code>hoge(a, b);</code>でよいのでしょうか。</p>
