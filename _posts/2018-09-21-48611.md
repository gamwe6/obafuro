<p>前と後ろの文字列を指定して、その間にある文字列を抜き出したいです。<br>
例</p>

```
const str = "あいうえおかきくけこさしすせそ";
const before = "あいうえお";
const after = "さしすせそ";
//出力: "かきくけこ"
```
