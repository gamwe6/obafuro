<p>プログラムは以下のとおりです。</p>

<pre><code>#! /usr/bin/env python
# -*- coding: shift_jis -*-
import numpy as np
import matplotlib.pyplot as plt
y, c, x1, x2 = np.loadtxt('data.txt', unpack=True)
color_mapper = np.vectorize(lambda x: {0: 'red', 1: 'blue'}.get(x))

plt.hlines(y, x1, x2, colors=color_mapper(c))
plt.margins(0.1)

plt.grid()
plt.show()
</code></pre>
