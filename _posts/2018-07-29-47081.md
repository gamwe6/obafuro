<p>Visual Studio 2017で括弧が補完された後で改行してもインデントが挿入されません。<br>
どうすればよいのでしょうか?</p>

<p>私の環境では、波括弧を入力すると自動的に補完され、下のようなカーソル位置になります。 <br>
(<code>|</code>がカーソルを表しています)</p>

```
if (a == "a") {|}
```

<p>この状態でEnterを押すと</p>

```
if (a == "a") {
|}
```

<p>というようになってしまいます。</p>

<p>これを</p>

```
if (a == "a") {
    |
}
```

<p>というように改行したときに自動的にインデントが入るようにしたいのですがどうすればよいのでしょうか?</p>

<p>ちなみにインデント幅は4スペースです。<br>
言語はC/C++を想定しています</p>
