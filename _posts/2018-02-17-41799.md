<p>ASP.NET MVC で Model のList型の値を JavaScript でJsonデータとして参照したいので、<br>
色々なサイトを参考に以下のように記述しました。</p>

<p>↓ View.cshtml の中へ記述</p>

<pre><code>&lt;Script type="text/javascript"&gt;
  @{
     var value = Newtonsoft.Json.JsonConvert.SerializeObject(Model.employees);
  }
  var employees = @value;
&lt;/script&gt;
</code></pre>

<p>↑  ↑  ↑<br>
JavaScriptのメンバ変数で受ける最後の行で、エラーが出ます。<br>
エラーは「SCRIPT1028: 識別子、文字列または数がありません」<br>
これを回避するには、どうしたらよいでしょうか。</p>

<p>メンバ変数の中身を確認すると、以下のように表示されます。</p>

<pre><code>var employees = [{&amp;quot;year&amp;quot;:&amp;quot;2001&amp;quot;,&amp;quot;month&amp;quot;:&amp;quot;03&amp;quot;},{&amp;quot;year&amp;quot;:&amp;quot;2002&amp;quot;,&amp;quot;month&amp;quot;:&amp;quot;04&amp;quot;},{&amp;quot;year&amp;quot;:&amp;quot;1999&amp;quot;,&amp;quot;month&amp;quot;:&amp;quot;10&amp;quot;}];
</code></pre>

<p>&quot; を「"」に変換したら、正しい値が入っているように思います。</p>

<pre><code>var employees = [{"year":"2001","month":"03"},{"year":"2002","month":"04"},{"year":"1999","month":"10"}];
</code></pre>

<p>IE11の問題なのでしょうか。原因や回避方法をご存知でしたら教えてください。<br>
よろしくお願いいたします。</p>
