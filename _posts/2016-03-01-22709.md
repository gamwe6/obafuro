<p>私は数学が大の苦手なので、見当違いの質問かもしれませんが・・</p>

<p>"プログラマの数学" (ISBN4-7973-2973-4)の中で<code>0からnまでの整数の和は(n * (n + 1)) / 2 に等しい</code>ということを証明するために数学的帰納法を使っています</p>

<p>基底の証明<code>G(0)</code>が成り立つことは実際に計算すればわかります</p>

<p><code>(0 * (0 + 1)) / 2 = 0   //確かに計算すると答えが0になる</code> </p>

<p>この中の帰納の証明の中でG(k) = <code>0 + 1 + ••• + k = (k * (k + 1)) / 2</code> が成り立つと<strong>仮定</strong>して話を進めていますが、<br>
これはあくまでも<strong>仮定</strong>であり、G(k)が成り立ったからといってG(k+1)が成り立つとは限らないのではないのでしょうか?</p>

<p>G(0)が成り立ったからといってG(1263712)が成り立つことは分からないのではないかと思います</p>

<p>数学的帰納法での証明は実際に正しいかどうかはわからないのでしょうか?</p>

<p><code>G(k) = 0 + 1 + ••• + k = (k * (k + 1)) / 2</code></p>

<p><code>G(k + 1) = 0 + 1 + ••• + k + (k + 1) = ((k + 1) * (k + 1) + 1) / 2</code></p>
