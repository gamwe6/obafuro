<p>c/c++で文字列への変換無しで小数点以下の桁数を求めたいです。  <br>
e.g)<br>
1.0 -> 0<br>
1.1 -> 1<br>
0.001 -> 3</p>

<pre><code>float x= 0.4002;
int i=0;
while( x != floor(x) )
{
  x*=10;
  ++i;
}
</code></pre>

<p>一応こんな方法は思いついたのですが、<br>
浮動小数だと正確に10倍されずに正しい桁数が出ないことが判明したので 正確に小数点以下の桁数を求める方法があれば教えてください。</p>
