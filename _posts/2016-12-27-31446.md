<p>以下のようなテーブル構造があります。</p>

<pre><code>CREATE TABLE `products` (
`uid` int(11) NOT NULL,
`item_code` varchar(255) NOT NULL,
`stock` int(11) NOT NULL,
`modified` datetime NOT NULL,
`created` datetime NOT NULL,
UNIQUE KEY `unique_keys` (`uid`,`item_code`)
)
</code></pre>

<p>uid:5, item_code:b-testのレコードがある状態で</p>

<pre><code>$test = array(
    (int) 0 =&gt; array(
        'item_code' =&gt; 'b-test',
        'uid' =&gt; '5'
    ),
    (int) 1 =&gt; array(
        'item_code' =&gt; 'w-test',
        'uid' =&gt; '5'
    )
);

$this-&gt;Product-&gt;saveAll($test);
</code></pre>

<p>このテーブルに対して上記のようなコードを実行すると</p>

<pre><code>Integrity constraint violation: 1062 Duplicate entry '5-b-test' for    key 'unique_keys'
</code></pre>

<p>と怒られてupdateになりません。<br>
cakephpでは複合ユニークキーの保存と更新は出来ないのでしょうか。</p>
