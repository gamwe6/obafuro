<p>下記で、ディレクトリの選択ダイアログが表示され、testメソッドでディレクトリ内のファイル名を取得することはできたのですが、<br>
選択したディレクトリまでの絶対パスは取得する方法はないでしょうか？</p>

<p>◆HTML</p>

<pre><code>&lt;input type="file" id="dirselect" onChange="test(this.files)" webkitdirectory directory /&gt;
</code></pre>

<p>◆Javascript</p>

<pre><code>function test(files) {
    var i;
    var res = document.getElementById("res");
    res.innerHTML = "";
    for (i=0; i&lt;files.length; i++) {
        res.innerHTML += files[i].fileName + "&lt;br /&gt;";
    }
}
</code></pre>
