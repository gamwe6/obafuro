<p>UnsafeMutablePointer型を何とか利用して<br>
構造体へのキャストもできるのではと考えたのですが、<br>
下記のようにした場合、<br>
(1)：構造体を初期化<br>
(2)：構造体のポインタを取得<br>
その後、(3)のようにしてみたもののmyptrにdataが設定されませんでした。<br>
何か良い方法はないのでしょうか。</p>

<h1>C言語のヘッダーファイル</h1>

<pre><code>typedef struct {
u_char test1;
u_char test2;
u_char test3[2];
} TEST_STR;
</code></pre>

<h1>swiftソース</h1>

<pre><code>func casttest(data: NSData) -&gt; UnsafeMutablePointer&lt;TEST_STR&gt; {
    var test = TEST_STR()   // (1)
    var myptr = withUnsafeMutablePointer(&amp;test){$0}   // (2)

    // myptr.memory.test1　で参照できた（初期化しているので値は0）

    data.getByte(&amp;myptr, length: sizeof(TEST_STR)    // (3)

    return myptr
}
</code></pre>
