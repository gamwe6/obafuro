<p>タイトルの件、asp.net で作成した画面の検索ボタンをクリックすると<br>
SQL Server 2016のデータベースのデータを検索し、検索結果をCSVでダウンロードする<br>
機能があります。</p>

<p>ただ、この検索処理が条件によっては重いため、検索中断等のボタンを設けて<br>
サーバサイドの検索処理の中断機能を実装できないか検討中です。</p>

<p>何か方法はありますでしょうか？<br>
検索処理を非同期で実装しないと実現不可能でしょうか。</p>

<p>.net framework は4.6.2になります。</p>
