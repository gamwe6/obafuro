<p>「新明解C++」という参考書で勉強してますが、派生についての記載がないので質問します。</p>

<ol>
<li>virtualをすると再定義することが出来ると知ったのですが、newした時にデフォルトコンストラクタの挙動はどうなるのでしょうか？</li>
<li><p>どっちの型でもnew 出来ますがどっちを利用すべきなんでしょか？<br>
例</p>

<pre><code>base *b = new base;
base *bb = new derive;
</code></pre></li>
<li><p>また、デスストラクタの挙動はどうなるのでしょうか？</p></li>
</ol>

<hr>

<pre><code>class base {
public:
    virtual void pri() {
        std::cout &lt;&lt; "base\n";
    }
};

class derive : public base {
public:
    void pri() {
        std::cout &lt;&lt; "virtual\n";
    }
};

int main() {

    base ba;
    derive de;

    base *b = new derive;
    base *bb = new base;
    derive *d = new derive;

    ba.pri();
    de.pri();
    std::cout &lt;&lt; "\n\n";
    b-&gt;pri();
    bb-&gt;pri();
    d-&gt;pri();

    _getch();
    return 0;
}
</code></pre>
