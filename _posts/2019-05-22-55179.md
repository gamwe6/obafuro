<p>bootstrap4でサムネイルリストを作りましたが、そのCSSで<code>object-fit</code>に<code>cover</code>としたにも関わらず、画像の高さが<code>cover</code>になりません。どうすれば改善するのでしょうか?</p>

<pre><code>&lt;div class="container"&gt;

  &lt;div class="row"&gt;
    &lt;div class="col-6"&gt;
      &lt;div class="row"&gt;
        &lt;a href="#" class="theme-color-text w-100"&gt;
          &lt;img src="https://placeimg.com/600/500/animals" class="image"&gt;
        &lt;/a&gt;
      &lt;/div&gt;
    &lt;/div&gt;
    &lt;div class="col-6"&gt;
      &lt;div class="row"&gt;
        &lt;a href="#" class="theme-color-text w-100"&gt;
          &lt;img src="https://placeimg.com/150/50/animals" class="image"&gt;
        &lt;/a&gt;
      &lt;/div&gt;
    &lt;/div&gt;

&lt;/div&gt;
</code></pre>

<p><strong>CSS</strong></p>

<pre><code>.text {
  position: relative;
}

.text::before {
  margin-top: 10px;
  display: block;
  content: "";
  padding-top: 100%;
}

.image {
  width: 100%;
  margin-top: 10px;
  object-fit: cover;
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
}
</code></pre>
