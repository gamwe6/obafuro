<pre><code>import Foundation


DispatchQueue.main.async {
    print("A")
}

print("B")
</code></pre>

<p>上記をプレイグラウンドで試行したとろころ、毎回</p>

<pre><code>B
A
</code></pre>

<p>と表示されました（試した回数は5回ほど）</p>

<p>さらに、</p>

<pre><code>import Foundation

for _ in 1...10 {
    DispatchQueue.main.async {
        print("A")
    }

    print("B")
}
</code></pre>

<p>を試したところ出力は、</p>

<pre><code>B
B
B
B
B
B
B
B
B
B
A
A
A
A
A
A
A
A
A
A
</code></pre>

<p>でした。</p>

<p>非同期で動くので、BとAが交互に出力される場合があってもおかしくない気がするのですが、なぜBが先に全部表示されるのでしょうか？</p>
