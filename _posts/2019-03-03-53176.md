<p>テストケースが通りません。<br>
入力結果を何個か確認したところあってそうなのですがどこがテストケースにひかかっているか不明です。<br>
解説いただければ幸いです。</p>

<p>▼問題文<br>
正整数 <br>
A,Bが与えられます。</p>

<p>Aも Bも割り切る正整数のうち、N番目に大きいものを求めてください。</p>

<p>なお、与えられる入力では、<br>
Aも Bも割り切る正整数のうち N番目に大きいものが存在することが保証されます。</p>

<p>制約<br>
入力は全て整数である。<br>
1≤A,B≤100<br>
Aも Bも割り切る正整数のうち、K番目に大きいものが存在する。<br>
N≥1</p>

<p>▼コード</p>

<pre><code>import java.util.ArrayList;
import java.util.Scanner;

public class Main {
    public static void main(String args[]) {
        Scanner scn = new Scanner(System.in);
        int a = scn.nextInt();
        int b = scn.nextInt();
        int n = scn.nextInt();

        ArrayList&lt;Integer&gt; list = new ArrayList&lt;&gt;();

        int i = 1;
        do {
            if (a % i == 0 &amp;&amp; b % i == 0) {
                list.add(i);
            }
            i++;
        } while (i &lt; a);

        System.out.println(list.get((list.size()-n)));
    }
}
</code></pre>
