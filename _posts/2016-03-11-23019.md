<p>タイトルのことを行いたいわけですが、やり方が何パターンかあります。</p>

<p>もっとも単純なものは</p>

<pre><code>//パターン1
var str:String!
if str == nil{
    str = "newValue"
}
</code></pre>

<p>次に、オプショナルバインディングを用いた時</p>

<pre><code>//パターン2
var str:String!
if let _ = str{
    //strがnilでないとき
}else{
    //strがnilのとき
    str = "newValue"
}
</code></pre>

<p>strがnilの時だけ値をいれたいので、else句以外は不要な上、冗長です。</p>

<p>次に、nil結合演算子を用いた時</p>

<pre><code>//パターン3
var str:String!
str = str ?? "newValue"
</code></pre>

<p>この場合、とても簡潔に書けるのですが、strを二回書いてるところがなんだかダサいのです。</p>

<p>やはりパターン１がよいのでしょうか。<br>
何か良い方法はありますか？</p>
