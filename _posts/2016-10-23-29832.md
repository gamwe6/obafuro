```
■フォルダ
|---■実行ファイル.py
|---■\__init__.py
|---■パッケージ
　　|---■\__init__.py
　　|---■モジュール1.py
　　|---■モジュール2.py
```

<p>のような関係で実行ファイルから</p>

```
from パッケージ　import モジュール1
```

<p>とした時、モジュール1の中の</p>

```
import モジュール2
```

<p>の文が上手く行かず</p>

```
ImportError: No module named 'モジュール2'
```

<p>と出てしまい、コンピューターには疎いため困っています。<br>
プログラムはWindows10でSpyderのIPython上で動かしています。<br>
