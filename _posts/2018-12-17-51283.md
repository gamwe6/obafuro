<p>プログラム初心者です。<br>
下記のコードを実行したところWHEREのところでエラーが出てしまって困っております。<br>
nameで入れたい型はcharacter(16)になっております。<br>
よろしくお願い致します。</p>

<pre><code>sqlalchemy.exc.ProgrammingError: (psycopg2.ProgrammingError) 演算子が存在しません: character = bigint

<p>コード：</p>

<pre><code>import sqlalchemy.ext.declarative
import pandas as pd

url = 'postgresql+psycopg2://postgres:*****@127.0.0.1:****/test'

engine = sqlalchemy.create_engine(url)

table_name ='test2'
name = '123456789101234'
query = 'select * from {} WHERE test_code = {}'.format(table_name, name)
df = pd.io.sql.read_sql(query, engine)
print('#############')
print(df)
