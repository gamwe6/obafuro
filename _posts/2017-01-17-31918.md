<p>Python3(3.5.2)で引数を1つとり、その引数に応じてfor文のステップ数を変更する方法をご教示ください。<br>
具体的には以下動作です。(以下は考えの説明をするための疑似コードです。実際には機能しません)</p>

<p>コマンドラインにて<br>
python ./test.py 100 <br>
と実行した場合</p>

<p>引数が50までは以下のような1ずつ増分するfor文処理を実施します。</p>

```
for i in range(0, args[0], 1): #50まで
    *hogehoge*
```

<p>50からは以下のような0.5ずつ増分するfor文処理を実施します。</p>

```
for i in range(50, args[0], 0.5):
    *hogehoge*
```

<p>つまりこの例では、<em>hogehoge</em>処理は、全部で150回実施されます。<br>
(0から50が50回、50から100が100回)<br>
このとき、<em>hogehoge</em> 部分の処理は同じです。</p>

<p>if文で分割すれば処理可能ですが、<em>hogehoge</em>部分の処理が同じであり、<br>
可能ならば1つのfor文で簡潔に表現したいと考えております。<br>
ご教示ください。</p>
