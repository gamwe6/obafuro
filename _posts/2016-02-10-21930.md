<p>以下のコードで</p>

<pre><code>function sendMail() {
    window.plugins.webintent.startActivity (
        {
            action: window.plugins.webintent.ACTION_VIEW,
            url: 'mailto:メールアドレス'
        },       
        function () {},
        function () {alert ('Failed to open URL via Android Intent');}
    );
}
</code></pre>

<p>以下のエラーが出ます。</p>

<blockquote>
  <p>Uncaught TypeError: Cannot read property 'webintent' of undefined</p>
</blockquote>

<p>Cordovaのwebintentは有効にしてあります。<br>
以下も指定してあります。</p>

<pre><code>&lt;script src="components/loader.js"&gt;&lt;/script&gt;
&lt;link rel="stylesheet" href="components/loader.css"&gt;
</code></pre>

<p>何か設定が足りないのかもしれません。<br>
ご教授お願いいたします。</p>

<p>すみません、解決しました。<br>
config.xmlに以下を追加しました。mailではなくてmailtoでした。</p>

<p><code>&lt;allow-intent href="mailto:*" /&gt;</code></p>
