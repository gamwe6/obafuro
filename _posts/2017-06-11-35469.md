<p>上記のような辞書のvalue部分が配列になっているJSONファイルがあります。</p>

<pre><code>import json

f = open("jsonFile.json")
data = json.load(f)
datas = data

values = datas.values()

for valueList in values:
    for value in valueList:
            print('&lt;li&gt;&lt;a href=\"{0}\" target=\"_blank\"&gt;&lt;/a&gt;/li&gt;'.format(value))
