<p>ネットワーク上のノードを表すオブジェクトに，メソッド呼び出しでアクセスするようなコードの書き方はありませんでしょうか？</p>

<p>例えば，Python で書くと，</p>

<pre><code>class Node:
    def __init__(self, ip, port):
        self.ip = ip
        self.port = port
        self.hoge = 'hogehoge'

    def do_something(self):
        n = Node('127.0.0.1', 12345)
        self.hoge = n.get_hoge()

    def get_hoge():
        return self.hoge
</code></pre>

<p>Java で書くなら，</p>

<pre><code>class Node {
    private String ip;
    private int port;
    private String hoge;

    public Node(String ip, int port) {
        this.ip = ip;
        this.port = port;
        this.hoge = "hogehoge";
    }

    public doSomething() {
        Node n = new Node('127.0.0.1', 12345);
        this.hoge = n.getHoge();
    }

    public getHoge() {
        return this.hoge;
    }
}
</code></pre>

<p>このような書き方ができるクラスを作りたいです．<br>
要するに，相互に通信を行う <code>Node</code> クラスからソケット API のコードを隠蔽したいということです．</p>

<p>言語は問いません．<br>
何かいい書き方はありませんでしょうか？<br>
