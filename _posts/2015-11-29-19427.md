<p>結合でデータを取ろうとしているのですが、以下のような例外が起こります<br>
この例外を回避する方法を知りたいです。</p>

<p>＜例外が状況＞<br>
T_estimateにデータがある<br>
T_estimate_detailsにデータがない</p>

<p>※両方データがある場合は例外は発生していません。</p>

<p>【実行しているJPQL】</p>

<pre><code>select 
    new sel.T(
    t_e.esti_rep_no
    , t_ed.esti_rep_details_no) 
    from 
    T_estimate t_e left join t_e.t_estimate_details t_ed
</code></pre>

<p>【結果格納クラス】<br>
    package sel;</p>

<pre><code>public class T {
    private int esti_rep_no;    
    private int esti_rep_details_no;

        // アクセサメソッドは省略
}
</code></pre>

<p>【スタックトレース】</p>

<pre><code>java.lang.IllegalArgumentException: org.hibernate.QueryException: could not instantiate class [sel.T] from tuple
</code></pre>
