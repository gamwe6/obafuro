<p>python初心者・独学者です。ある程度のことはできるようになったのですが、以下の方法が全くつかめずに頭を抱えています。<br>
どなたかよろしくお願いいたします。</p>

```
list_a = [
    { "a" : "住所A" , "b" : "成人" , "c" : "女" , "d" : "いる" } ,
    { "a" : "住所B" , "b" : "成人" , "c" : "男" , "d" : "いる" } , 
    { "a" : "住所C" , "b" : "小児" , "c" : "男" , "d" : "いる"} ,
    { "a" : "住所D" , "b" : "成人" , "c" : "男" , "d" : "いない"} ]
```

<p>のような二次元配列のリストがある場合、<code>"b" == "小児"</code> の場合は、そのリストを削除して以下の新たなリストを作りたいと思っています。</p>

```
list_b = [
    { "a" : "住所A" , "b" : "成人" , "c" : "女" , "d" : "いる" } ,
    { "a" : "住所B" , "b" : "成人" , "c" : "男" , "d" : "いる" } , 
    { "a" : "住所D" , "b" : "成人" , "c" : "男" , "d" : "いない"} ]
```

<p>つまり、二次元配列リスト（list_a）内の、特定のリスト<code>({ "a" : "住所C" , "b" : "小児" , "c" : "男" , "d" : "いる"})</code>内の、特定の要素<code>("b" : "小児")</code>に当てはまる場合は、リスト全体から、このリストのみを外したいのです。</p>

<p><code>for in range ()</code>文を使ってやってみたのですが、なぜか<code>out of range</code>のエラーが出てしまい、途中でストップしていまいます。おそらく、リストが削除されると、その分だけ<code>range()</code>の数が減ってしまっていくことが原因と思われますが、それを回避する方法が分かりません。</p>

<p>特に<code>for</code>文にこだわっているわけではないので、他にも方法があるのであれば、教えていただけると嬉しいです。</p>

<p>非常にわかりにくい質問となってしまいましたが、御教授いただけると幸いです。</p>

<p>実際に試したコードは以下です。</p>

```
list_a = [
        { "a" : "住所A" , "b" : "成人" , "c" : "女" , "d" : "いる" } ,
        { "a" : "住所B" , "b" : "成人" , "c" : "男" , "d" : "いる" } ,
        { "a" : "住所C" , "b" : "小児" , "c" : "男" , "d" : "いる"} ,
        { "a" : "住所D" , "b" : "成人" , "c" : "男" , "d" : "いない"}
        ]

for i in range ( 0 , len(list_a) ) :
    if list_a[i]["b"] == "小児" :
        list_b == list_a.remove (list_a[i]) 

print (list_b)
```

<p>if文の後にelse文が載っていませんが、これを付けるとエラーが出てしまうか、breakすると、Noneが帰ってきます。そもそも、for文でこの問題を解決できるのかどうかかが、分かりません…。</p>
