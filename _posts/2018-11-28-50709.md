<p>お世話になります。</p>

<p>大変情けない話で申し訳ないのですが、C#にて自作のDLLを作成し、VBA側で使用するまでは<br>
たどり着いたのですが、C#側でStaticで宣言したクラスをDLLで呼び出す文法が解りません。</p>

<pre><code>   public class Class1
    {
        public Class1()
        {

        }

        public string getMsg()
        {
            return "Hello World.";
        }
    }
</code></pre>

<p>↓↓↓↓↓↓↓↓</p>

<pre><code>Public Sub test()
    Dim a As Sample.Class1
    Set a = New Sample.Class1
    MsgBox a.getMsg
End Sub
</code></pre>

<p>これは解りました。</p>

<pre><code>public static class Class1
{

    public static string getMsg()
    {
        return "Hello World.";
    }
}
</code></pre>

<p>↓↓↓↓↓↓↓↓</p>

<p>これが解りません。</p>
