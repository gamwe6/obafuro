<p>なぜエラーになるのでしょうか？</p>

<blockquote>
  <p>ActionView::Template::Error (SQLite3::SQLException: near "all": syntax<br>
  error: SELECT "items".* FROM "items" WHERE (h >= 0) AND (w >= 0) AND<br>
  (d >= 0) AND (weight >= 0) AND (all >= 0) ORDER BY "items"."price"<br>
  ASC):</p>
</blockquote>

<pre><code>@items = Item.where('h &gt;= ?',height).where('w &gt;= ?',width).where('d &gt;= ?',depth).where('weight &gt;= ?',weight).where('all &gt;= ?',total).order(price: :asc)
</code></pre>

<p>ここで、</p>

<pre><code>.where('all &gt;= ?',total)
</code></pre>

<p>を省くと、正常に動きます。</p>

<p>データのカラムは以下の通りです。</p>

<pre><code>class CreateItems &lt; ActiveRecord::Migration[5.1]
  def change
    create_table :items do |t|
      t.integer :h
      t.integer :w
      t.integer :d

      t.timestamps
    end
  end
end
</code></pre>

<p>と</p>

<pre><code>class AddColumnsToItems &lt; ActiveRecord::Migration[5.1]
  def change
    add_column :items, :name, :string
    add_column :items, :price, :integer
    add_column :items, :weight, :integer

  end
end
</code></pre>
