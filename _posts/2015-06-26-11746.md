<p>2番目の数値が小さい順に、名前順でソートしたい。<br>
ファイラーのようにフォルダの後にファイルが並ぶ。</p>

<p>配列の1番目の値が名前です。<br>
二番目の値は0がフォルダ、1がファイルとして付けます。<br>
次のコードを実行すると2番目の値に沿ってソートはされますが、配列の内容によっては名前順にソートされません。</p>

<pre><code>var ary = [
    ['あ0', 0],
    ['b', 0],
    ['c.txt', 1],
    ['た', 0],
    ['e', 0],
    ['ぜ.txt', 1],
    ['a', 0],
    ['a.txt', 1],
    ['d', 0],
    ['あ', 0],
    ['c', 0],
    ['b.txt', 1],
];

ary.sort(function(a, b) {
  return (a[1] - b[1]) || (a[0] &gt; b[0]);
});

for (var i = 0; i &lt; ary.length; i++) {
  console.log('%d: %s', ary[i][1], ary[i][0]);
}
</code></pre>

<p>次の配列に差し替えるとうまくソートできました。</p>

<pre><code>var ary = [
    ['あ0', 0],
    ['b', 0],
    ['c.txt', 1],
    ['た', 0],
    ['e', 0],
    ['ぜ.txt', 1],
    ['a', 0],
    // ['a.txt', 1],
    ['d', 0],
    ['あ', 0],
    ['c', 0],
    ['b.txt', 1],
];
</code></pre>
