<p>　C系の文法を持つ言語については以下のような記述が可能であると思います。</p>

<pre><code>while (foo) if (hoge) {
    ...
}
</code></pre>

<p>　これは、こちらの書き方よりもブロックネストを抑えることができるのが利点ですが、</p>

<pre><code>while (foo) {
    if (hoge) {
        ...
    }
}
</code></pre>

<p>可読性的には「もしかしたら問題があるかもしれない」と考えています。</p>

<p>ただ、個人的にはネストを抑える効果は大きいと考えています。</p>

<p>　<br>
　この書き方は、一般的には問題がありますか？</p>
