<p>文字列がある場合には、欠損値np.nanにしたいと考えています。<br>
現在は、次のようなコードを考えています。</p>

<pre><code>import pandas as pd
import numpy as np

df = pd.DataFrame({'x': [3, 'TV', np.nan]})

#文字列がある場合には、欠損値とする。
df['x'].applymap(lambda row: np.nan if type(row) is str, axis=1)
</code></pre>
