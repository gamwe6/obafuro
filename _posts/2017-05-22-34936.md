<p>[css]</p>

<pre><code>@charset "UTF-8";

*{
  margin: 0;
  padding: 0;
  text-align: center;
}

/*中の幅に合わせて縦並びにしたい*/
.container{
    display: flex;
    background: skyblue;
    justify-content: space-between;
    flex-direction: column;
}

.item{
    background: tomato;
    flex: 0 1 auto;
}
</code></pre>
