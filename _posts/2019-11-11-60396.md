<pre><code>import csv

    print(row)

</code></pre>

<p>現在このような形でcsvファイルを読み込んでおり、読み込みには成功しています。<br>
ここからE列にいくつ１０以上数値があるかをカウントしたいと思っております<br>
そこで</p>

<pre><code>import csv
f = open("test2.csv", "r", )
reader = csv.reader(f)
count = 0
for row in reader:
    colE = int(row[5])
    if colE &gt;= 10:
        count += 1
        print(count)


f.close()
</code></pre>
