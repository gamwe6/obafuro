<p>ruby on railsで制作した写真投稿アプリをポートフォリオ用にnetlifyを使用して見れるようにしたいのです。</p>

<p>ターミナルで</p>

<p>git init<br>
git add .<br>
git commit -m "pixcelgram"<br>
git push -u origin master</p>

<p>とした後にnetlify <a href="https://app.netlify.com" rel="nofollow noreferrer">https://app.netlify.com</a>　にて</p>

<p>New site from git　のボタンを押して</p>

<p>左下のGitHubボタン押すと私のgithubに飛んでpixelgramを選ぶと</p>
