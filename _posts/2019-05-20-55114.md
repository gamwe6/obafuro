<p>以下の様に書いても、008の次に001が表示され、009以降の表示が出来ないのですが<br>
どの様にすれば良いのでしょうか</p>

<pre><code>@echo off
SetLocal EnableDelayedExpansion
SET NUM=1
:Loop
IF !NUM! LEQ 99 ( IF !NUM! LEQ 9 (SET NUM=00!NUM!) else (SET NUM=0!NUM!) ) else (SET NUM=!NUM!)
ECHO !NUM!
SET /A NUM+=1
GOTO :Loop
EndLocal
EXIT /B
</code></pre>
