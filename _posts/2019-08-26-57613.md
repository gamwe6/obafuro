```
hash = {"1.3"=&gt;"53.0", "3.4"=&gt;"50.30", "2.5"=&gt;"1.600", "7.4"=&gt;"63.0", "4.9"=&gt;"103.400", "3.2"=&gt;"13.400"}
```

<p>a = 3.0<br>
上記のような場合にkeyがa（3.0）以上のvalue値を返したいのですがなかなか上手くいきません。</p>

<p>上の場合だと</p>

```
{"3.4"=&gt;"50.30","7.4"=&gt;"63.0","4.9"=&gt;"103.400", "3.2"=&gt;"13.400"}
```

<p>もしくはvalue値のみで</p>

```
{50.30, 63.0, 103.400, 13.400}
```

<p>と返したいです。<br>
何かいい方法をご存知の方は教えていただけるとありがたいです。</p>

<p>また、hashでkeyもvalue文字列の場合にhashの形は保ったまま文字列 → 数値に変更する方法はあるのでしょうか。</p>
