<p>Swift2からSwift3へのConvertを行なったところ、以下のエラーが出て解決できません。</p>

<pre><code>Cannot convert value of type 'UnsafePointer&lt;xmlChar&gt;' (aka 'UnsafePointer&lt;UInt8&gt;') to expected argument type 'UnsafePointer&lt;_&gt;'
</code></pre>

<p>該当コードは以下の通りです。<code>String(cString: UnsafePointer&lt;CChar&gt;(char!))</code>のところでエラーが出ます。</p>

<pre><code>static func fromXmlChar(_ char: UnsafePointer&lt;xmlChar&gt;?) -&gt; String? {
    if char != nil {
        return String(cString: UnsafePointer&lt;CChar&gt;(char!))
    } else {
        return nil
    }
}
</code></pre>
