date: 2017-08-30 02:17:32
categories: java security
---
<p>maven central など nexus repository を利用することで、いろいろな jar ファイルを依存関係として追加できます。ふと、ここでアップロードされている jar ファイルが、対象のソースコードからビルドされているのかを確かめる方法があるのか、と疑問に思いました。というのも、ソースコードをビルドしてデプロイする間に、悪意の第三者が入りこんで、 jar の成果物を置き換えることが、できてしまうのではないか、と考えたからです。</p>

<p>質問:</p>

<ul>
<li>ビルドされたであろうソースコードがわかっている場合に、対象の jar ファイルが、確かにそのソースコードをビルドした結果得られるものであると検証することはできますか？</li>
</ul>
