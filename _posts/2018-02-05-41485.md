<p>32～127のASCIIコード(16進、10進、キャラクタ)を表示します。<br>
(0～31番は画面に表示できない文字なので、表示しません)</p>

<pre><code>#include &lt;stdio.h&gt;

main()
{
    int x, y;       /*ループカウンタ*/
    char c;         /*キャラクタ番号*/

    for(x = 2; x &lt; 8; x++)
        printf("16: 10:c | ");
    printf("\n");
    for(x = 2; x &lt; 8; x++)
        printf("---------+-");
    printf("\n");

    for(y = 0; y &lt; 16; y++)
    {
        for(x = 2; x &lt; 8; x++)
        {
            c = x * 16 + y;
            printf("%2x:%3d:%c | ", c, c, c);
        }
        printf("\n");
    }
}
</code></pre>

<p>こちらは(株)アンクの｢Cの絵本｣という参考書からなのですが、上記のプログラムだと実行時にエラーが出ます。何故なのでしょうか。</p>
