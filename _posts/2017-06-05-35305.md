<p>Pythonで以下のようなデコレータを定義したとします。</p>

<pre><code>def trace(f):
    @functools.wraps(f)
    def wrapper(*args, **kwargs):
        print('Enter Func')
        try:
            ret = f(*args, **kwargs)
        except:
            print('EXCEPTION')
            raise
        print('Leave Func')
        return ret
    return wrapper
</code></pre>

<p>このデコレータは通常の関数であれば期待通り動作します。</p>

<pre><code>def func():
    print('Hello')

func = trace(func)
func()

&lt;&lt; Enter Func
&lt;&lt; Hello
&lt;&lt; Leave Func
</code></pre>

<p>しかし、ジェネレータ関数に対しては、期待通り動作しません。</p>

<pre><code>def generator_func():
    print('Start Generator')
    for i in range(3):
        yield(i)

generator_func = trace(generator_func)
for x in generator_func():
    print(x)

&lt;&lt; Enter Func
&lt;&lt; Leave Func
&lt;&lt; Start Generator
&lt;&lt; 0
&lt;&lt; 1
&lt;&lt; 2
</code></pre>

<p>期待する出力は下記の通りです。このような出力を得るためのデコレータは定義可能でしょうか？</p>

<pre><code>&lt;&lt; Enter Func
&lt;&lt; Start Generator

</code></pre>
