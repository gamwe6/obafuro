<p>webpack.config.dev.js へ</p>

<pre><code>const I18nPlugin = require("i18n-webpack-plugin");
const langJa = require("./ja.json");
・・・
module: {
  loaders: [
    {
      test: /\.(ts|tsx)$/,
      include: paths.appSrc,
      loader: "ts"
    },
  ]
},
・・・
plugins: [
      new I18nPlugin(langJa),
    ],
</code></pre>

<p>と記載し、app.tsxに</p>

<pre><code>console.log(__("Hello World"));
</code></pre>

<p>としたところ、</p>

<pre><code>error TS2304: Cannot find name '__'.
</code></pre>

<p>とエラーが返ってきます。<br>
コンソール上には こんにちは と表示されています。</p>

<p>おそらく i18n-webpack-plugin の処理が行われる前にTypeScriptが変換しているからだと思うのですが、TypeScriptよりも先にi18n-webpack-pluginの処理を行う方法などありますでしょうか</p>
