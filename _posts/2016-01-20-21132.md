<pre><code># coding: utf-8

import Image as img
import numpy as np
import matplotlib.pyplot as plt

img = np.array(img.open("a.png"))

width  = img.shape[0]
height = img.shape[1]

for i in range(height + 1) :
    for j in range(width + 1) :
        img[i][j] = img[i][width] - img[i][j]

plt.imshow(img)
plt.show()
</code></pre>

<p>python初心者なのですが画像を左右反転させるコードを書いたのですが, </p>
