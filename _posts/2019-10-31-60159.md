<p>ArduinoUNOでIDE1.8.7を使用しています。</p>

<p>表題の計算を以下のように行いました。</p>

<pre><code>int Dtoi(String s,int idx){
　int sum = 0,x,sl;
　sl = s.length();
　for(int i = 0;i &lt; sl;i++){
  　　if(s.charAt((idx + i)) &gt; 55){
　　　x = s.charAt((idx + i)) - 55;
    　　}else{
      　　　x = s.charAt((idx + i)) - 48;
    　　}
    　　Serial.print(sum);
    　　Serial.println();
    　　sum += x * pow(16,(sl-(i+1)));
  　}
  　Serial.print(sum);
  　return sum;
}
</code></pre>

<p>メインで</p>

<pre><code>int a;
a = Dtoi("111",0);
</code></pre>

<p>のように実行すると<br>
結果が</p>

<pre><code>0
255
271
272
</code></pre>

<p><code>a = Dtoi("11",0);</code>だと結果が</p>

<pre><code>0
15
16
</code></pre>

<p><code>a = Dtoi("1",0);</code>だと結果が</p>

<pre><code>0
1
</code></pre>

<p>と表示されます。</p>
