<p>本日学び始めた初学者です。</p>

```
home="america"
if home =="america":
    print("Hello,america!")
else:
    print("Hello,world")
```

<p>というコードを通したいのですが、<br>
必ずエラーが返ってしまいます。<br>
①そもそも、テキストエディタ（サクラエディタ）に作成したコードを、<br>
python 3.6.5 Shellにコピペしてはいけないのでしょうか。</p>

<p>②1行ずつコピペしても（打っても）else: を打ってエンターキーを押すと<br>
必ずエラーとなってしまいます。（invalid syntax）</p>

<p>上記はどうしてなのでしょうか。<br>
基本的なところで間違っているのかもしれませんが、どう調べてもわからず。。</p>
