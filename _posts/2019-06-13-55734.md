<p>下記結果を返すSQLがある時</p>

<pre><code>SELECT id FROM (SELECT * FROM `s` WHERE `c_id` != 9) AS a
1
2
5
7
</code></pre>

<p>「上記結果何れにも該当しない」という条件を指定するにはどうすれば良いですか？</p>

<hr>

<p>やりたいことのイメージ</p>

<pre><code>SELECT * FROM `r` WHERE `t_id` != 1 かつ
SELECT * FROM `r` WHERE `t_id` != 2 かつ
SELECT * FROM `r` WHERE `t_id` != 5 かつ
SELECT * FROM `r` WHERE `t_id` != 7
</code></pre>

<p>そのまま書くと下記エラーとなります</p>

<pre><code>SELECT * FROM `r` WHERE `t_id` != (SELECT id FROM (SELECT * FROM `s` WHERE `c_id` != 9) AS a)
#1242 - サブクエリが2行以上の結果を返します。
</code></pre>
