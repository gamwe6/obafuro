<p>Design a class that stores a mathematical set of integers called MyClass . You may assume that the set will never have more than 100 elements. </p>

<p>• A default constructor that initializes a set to the empty set.<br>
• Overload the "^" operator to implement the set membership. Returns true if an<br>
element is in the set.<br>
• Overload the "+" operator to add an element to the set. Return the original set with<br>
the new element added.<br>
• Overload the "+" operator to implement the union of two sets. Returns a new set<br>
that contains all the elements of the both sets.<br>
• Overload the "&lt;&lt;" operator to print the set (in the format {1, 2, 3, 4})</p>

<p>こういう課題が出たのですがOverloadが何かということと何をすればいいのかが分かりません。<br>
