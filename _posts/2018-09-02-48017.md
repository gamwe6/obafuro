<p>例として、関数fは</p>

<pre><code>int f ((int, string))
</code></pre>

<p>のような型を持つとします。</p>

<p>実際に関数fを定義するとき、</p>

<pre><code>int f ((int x, string s) t) {
    return t.x;
}
</code></pre>

<p>のように、仮引数のタプルtは定義できます。</p>

<p>しかし、タプルtは大抵の場合、意味を持たず、名前が冗長になりがちです。<br>
なので、例えば、</p>

<pre><code>int f ((int x, string s)) {
    return x;
}
</code></pre>

<p>のように、関数fの仮引数でタプルの分解がしたいのですが、それは可能でしょうか？</p>
