<h2>Table</h2>

<pre><code>CREATE TABLE users (
  id INT NOT NULL AUTO_INCREMEN
  name VARCHAR(32),
);
</code></pre>

<h2>Model</h2>

<pre><code>class User &lt; ActiveRecord::Base
  attr_accessor :hoge

  def set_hoge
    self.hoge = "hogehoge"
  end
end
</code></pre>

<h2>実現したいこと</h2>

<p>railsコマンドラインにて</p>

<pre><code>&gt; user = User.take
&gt; user.set_hoge
&gt; user.attributes
</code></pre>

<p>とすると、<br>
{"id"=>1, "name"=>"Tom"}のように返されますが、</p>

<p>{"id"=>1, "name"=>"Tom", "hoge"=>"hogehoge"}<br>
のように返す方法が知りたいです!<br>
よろしくおねがいします。</p>
