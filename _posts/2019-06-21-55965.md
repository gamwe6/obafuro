<p>自分はC言語が得意じゃないため、皆さんの力貸してください。<br>
下記のRubyで作成したソースコードをC言語に変換したいです。</p>

<pre class="lang-rb prettyprint-override"><code>n = gets.to_i

ans = 0
i = 1
n.times {
  a = gets.to_i
  ans += (a-i).abs
  i = a
}

puts ans
</code></pre>
