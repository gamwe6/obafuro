<p>失礼します。<br>
EclipseやAPIを利用してWEBアプリを作成しています。<br>
URLに記載されたページに飛ぶとJSON形式のデータが表示されるのですが、可視化しやすいようにテーブルにすることはできないでしょうか？できればURLは固定でなくするのでなく、検索向けのURLにしたいです。<br>
JSONの書式は以下になります。</p>

```
[
    {
        "version": "1.5",
        "codename": "Cupcake"
    },
    {
        "version": "1.6",
        "codename": "Donut"
    },
```

<p>最終的に検索したページに飛びテーブル表示にしたいです。<br>
何卒宜しくお願いします。  </p>
