<p>2つのプログラムでカートの中身を表示するプログラムを作成しました。<br>
Cart.java <br>
Item.java<br>
しかし実行するとNullpointerExceptionとなってしまいます。改善策をご教授願いたいです。<br>
配列の代入が自信ないです。<br>
Cart.java</p>

<pre><code>public class Cart {
    Item[] items;
    int num=0,i;
    void addItem(Item x){
        items[num]=x;
        num++;  
    }
    void info(){
        for(i=0;i&lt;=num;i++){
            items[num].toString();
        }
    }

    public static void main(String[] args) {
        System.out.println(Item.getNumberOfInstances());
        Cart cart = new Cart();
        Item i1 = new Item("PC", 98000);
        Item i2 = new Item("Display", 40000);
        cart.addItem(i1);
        cart.addItem(i2);
        cart.info();
        System.out.println(Item.getNumberOfInstances());
    }
}
</code></pre>
