<p>クラス内のメンバ関数からクラス内の別のメンバ関数にアクセスしたいです<br>
自分は</p>

```
class X {
    public function a() {
        $x = 1234;
    }
    public function b() {
        print $x;//1234
    }

}
```

<p>のようなことを行いたいと思ってます。<br>
可能であればどんな書き方が最も正しいのかご教授ください</p>

<p>訳あってclass変数ではなく、関数内ではじめて定義された変数にアクセスしたいです</p>

<p>又、こういう処理が適切でない場合、皆様はclass内で定義される関数をどのようにして繋げて一まとめにしているのか</p>

<p>書き方の基礎を教えてくれると幸いです</p>
