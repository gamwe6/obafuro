<p>Python3で質問です。<br>
f(0)=1, f(1)=1, f(n)=f(n-1)+f(n-2), n>1と言う条件でnが1から20までの再帰を行いたいのですが</p>

<pre><code>def f(n):
    if n == 0 or n == 1:
        return 1
    else:
        return f(n-1)+f(n-2)
def main():
    for i in range(11):
        print(f(i))
main()
</code></pre>

<p>帰ってくる値は<br>
1<br>
1<br>
2<br>
3<br>
5<br>
8<br>
13<br>
21<br>
34<br>
55<br>
89</p>

<p>もともとのコードはこうなのですが、これをreturn時にlistに入れてreturnしたいのですがうまくいきません</p>

<pre><code>def f(n):
    L = []
    ans = []
    if n == 0 or n == 1:
        ans = 1
        L.append(ans)
        return L
    else:
        ans = f(n-1)+f(n-2)
        L.append(ans)
        return L
def main():
    for i in range(11):
        print(f(i))
main()
</code></pre>

<p>このように書くと上のような値が帰ってきません。</p>

<p>どのようにすればlistに入れてreturnさせてもうまくいきますか？<br>
お知恵を貸してください</p>
