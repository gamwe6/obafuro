<p>li1を昇順に並べ替えた時にli2[i]も,最初のli1[i]とひも付けしたように並べ替えて2次元配列にして欲しいです。</p>

<pre><code>li1=[1, 1, 2, 3, 4, 1, 1, 2, 3, 4, 1, 1, 2, 3, 4]

li2=[1, 10,40,70,80, 2,11,41,71,81,3,12,42,72,82]
</code></pre>

<p>結果は以下の通りになると思います。</p>

<pre><code>li1=[1, 1, 1, 1, 1, 1, 2, 2, 2, 3, 3, 3, 4, 4, 4]


li2=[[1,10,2,11,3,12],[40,41,42],[70,72],[80,82]]
</code></pre>

<p>よろしくお願いします。</p>
