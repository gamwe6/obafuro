date: 2018-08-08 15:13:33

<pre><code>$(function() {

  var duration = 300;

  $('#buttons1 button:nth-child(-n+4)')
    .on('mouseover', function() {
      $(this).stop(true).animate({
        backgroundColor: '#ae5e9b',
        color: '#fff'
      }, duration);


    })
    .on('mouseout', function() {
      $(this).stop(true).animate({
        backgroundColor: '#fff',
        color: '#ebc000'
      }, duration);


    });
});
</code></pre>

<p>HTML に変化がありません。文法的におかしいところはありますか。</p>
