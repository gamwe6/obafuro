```
list1 = ['12:00～15:00', '18:00～23:30', '12:00～15:00', '18:00～24:00']
```

<p>を</p>

```
list1 = ['12:00','15:00', '18:00','23:30', '12:00','15:00', '18:00','24:00']
```

<p>にしたいです。splitでやってみたんですが、リストは指定できないとエラーが出ます。何かやり方はありますか？</p>

```
list1 = ['12:00～15:00', '18:00～23:30', '12:00～15:00', '18:00～24:00']
split('～', list1)
```
