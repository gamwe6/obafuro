<p>これを動かすと</p>

<pre><code>&lt;?php

function raw_json_encode($input) {

    return preg_replace_callback(
        '/(?&lt;!\\\\)\\\\u(\w{4})/',
        function ($matches) {
            return mb_convert_encoding(pack('H*',$matches[1]),'UTF-8','UTF-16');
        },
        json_encode($input)
    );

}

$arr = array('Response'=&gt;array('Shipment'=&gt;array('shipId'=&gt;'1','test'=&gt;'1234/12/12'),
                               'Shipment'=&gt;array('shipId'=&gt;'2','test'=&gt;'1234/01/01')));

var_dump(raw_json_encode($arr, JSON_HEX_TAG | JSON_HEX_AMP | JSON_HEX_APOS | JSON_HEX_QUOT));
</code></pre>

<p>表示がこのようになります。</p>

<pre><code>"{"Response":{"Shipment":{"shipId":"2","test":"1234\\/01\\/01"}}}"
</code></pre>

<p>今私がほしいのが、この結果なんですが</p>

<pre><code>"{"Response":{"Shipment":{"shipId":"1","test":"1234/12/12"},"Shipment":{"shipId":"2","test":"1234/01/01"}}}"
</code></pre>

<p>どうやったらできますか？</p>
