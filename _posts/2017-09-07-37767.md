<p>jQueryを使って以下のようにしてみましたが、変化ありませんでした。</p>

<p>・javascript</p>

<pre><code>$(function() {
  $('#inputTextArea')
    .focusin(function(e) {
      $(this).css('height', '250px');
    })
    .focusout(function(e) {
      $(this).css('height', '80px');
    });
});
</code></pre>
