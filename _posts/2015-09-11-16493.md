title: 配列の中にあるオブジェクトの参照の仕方
date: 2015-09-11 01:48:46
categories: ruby
---
<pre><code>temp = [
  {
    :hoge =&gt; 1,
    :huga =&gt; []
  },
  {
    :hoge =&gt; 2,
    :huga =&gt; []
  },
]
</code></pre>

<p>この配列の「<code>hoge</code>が<code>1</code>のオブジェクトを参照する」方法はありますか？</p>

<p>イメージとしては以下のような操作を行いたいです。</p>

<pre><code>temp[{:hoge == 2}][:huga] &lt;&lt; 123
</code></pre>

<p>↓</p>

<pre><code>[
  {
    :hoge =&gt; 1,
    :huga =&gt; []
  },
  {
    :hoge =&gt; 2,
    :huga =&gt; [123]
  },
]
</code></pre>
