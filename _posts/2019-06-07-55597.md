<p><code>App.vue</code></p>

<pre><code>&lt;template&gt;
    &lt;div class="container"&gt;
        &lt;button @click="onClick"/&gt;
            Click
        &lt;/button&gt;
    &lt;/div&gt;
&lt;/template&gt;

&lt;script&gt;
import Vue from 'vue';
import { mapGetters } from 'vuex';

export default {
    methods: {
        onClick() {
            this.sample.values.push(1);
        }
    },
    computed: {
        ...mapGetters({
            sample: 'sample/getSample'
        })
    },
    watch: {
        sample: {
            handler: function(newVal, oldVal) {
                console.log(JSON.stringify(val));
                console.log(JSON.stringify(newVal));
            },
            deep: true
        }
    }
};
</code></pre>

<p><code>/stores/sample/index.js</code></p>

<pre><code>export const state = () =&gt; ({
    sample: {
        id: '',
        values: [1, 2, 3, 4, 5]
    }
});

const actions = {};
const getters = {
    getSample: state =&gt; {
        return state.sample;
    }
};
const mutations = {};

export default {
    namespaced: true,
    state,
    actions,
    getters,
    mutations
};
</code></pre>

<p>上記のような実装をしていて、App.vueはsampleの変更を検知した場合に、watchが走り、newVal、oldValをそれぞれ受け取ります。という想定でいたのですが、実際にこのコードを実行しても、</p>

<pre><code>{"id":"","values":[1,2,3,4,5,1]}
{"id":"","values":[1,2,3,4,5,1]}
</code></pre>

<p>というような表示しかされず、deepオプションを付与しても有効化されません。<br>
