ソースはこのような感じです。</p>

<pre><code>01: void Start () {
02:StartCoroutine (CoroutineSample());

11: 複数のelse if(他のキーを押したとき、Aなら左、Sなら下、Dなら右に移動する処理）
12:
13:　}//CoroutineSample()を閉じる括弧
</code></pre>

<p>ところがこれでビルドすると、</p>

<blockquote>
  <p><code>WaitWhile (!Input.anyKeyDown){ yield return 0; }</code>に(；が必要です)。</p>
