<p><strong>Chromeのコンソール画面でオブジェクトを表示させた時、先頭に表示されるのはコンストラクタ名？</strong><br>
・Hoge {}<br>
・hoge {site: "スタック"}</p>

<hr>

<pre><code>function Hoge() {
  console.log(this); //Hoge {}
}
var obj = new Hoge();
</code></pre>

<hr>

<pre><code>var service = {
 p: '1',
 hoge: function(site) {
   this.site   = "スタック";
 }
}
var target = new service.hoge();
console.log(target); //hoge {site: "スタック"}
</code></pre>

<hr>
