<p>例えば下の文で、</p>

<pre><code>#include&lt;iostream&gt;
#include&lt;cmath&gt;

float Myabs(float x);

int main(){
    float a=2.5;
    std::cout&lt;&lt;Myabs(a)&lt;&lt;std::abs(a);
    a=-5.25;
    std::cout&lt;&lt;Myabs(a)&lt;&lt;std::abs(a);
    return 0;
}

float Myabs(float x){return x&gt;0?x:-x;}
</code></pre>

<p>この文を下の2つの例のように変更すると<br>
例１</p>

<pre><code>#include&lt;iostream&gt;

float Myabs(float x);

int main(){
    float a=2.5;
    std::cout&lt;&lt;Myabs(a)&lt;&lt;std::abs(a);
    a=-5.25;
    std::cout&lt;&lt;Myabs(a)&lt;&lt;std::abs(a);
    return 0;
}
#include&lt;cmath&gt;
float Myabs(float x){return x&gt;0?x:-x;}
</code></pre>

<p>例２</p>

<pre><code>#include&lt;iostream&gt;
#include&lt;cmath&gt;

int main(){
    float a=2.5;
    std::cout&lt;&lt;Myabs(a)&lt;&lt;std::abs(a);
    a=-5.25;
    std::cout&lt;&lt;Myabs(a)&lt;&lt;std::abs(a);
    return 0;
}
float Myabs(float x);
float Myabs(float x){return x&gt;0?x:-x;}
</code></pre>

<p>どちらもエラーが出ます。宣言が無いために出るエラーとインクルードが無いために出るエラーです。<br>
これは、コンパイラから見てinclude文と宣言が同じ動作をすることを示していますか？</p>
