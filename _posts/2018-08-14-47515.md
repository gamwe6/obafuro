<p>C++で自己参照構造体というものがあると思います。（後述の二分探索木の例を参照）</p>

<p>この場合、p,l,rの型はNodeではなくてNode*です。<br>
これはNodeではいけないのでしょうか？</p>

<p>ネットで自己参照構造体をぐぐってみると、みんなポインタを使ってはいるものの、<br>
ポインタである必要性を説明している記事が見当たらなかったので不思議に思ってます。</p>

```
struct Node {
  int key;
  Node *p, *l, *r;
}
```
