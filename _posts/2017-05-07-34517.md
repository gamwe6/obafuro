<p><code>f()</code>で定義したローカル変数<code>b</code>の値を、ネスト関数<code>g()</code>から変更するにはどうすれば良いのでしょうか。<br>
<code>global a, b</code>ではうまくいかず、さりとて代案も思いつきませんでした。</p>

<pre><code>a = "global!!"

def f():
    global a
    print(a)
    b = "local 1"

    def g():
        global a
        print(a)
        print(b) # local variable 'b' referenced before assignment
        b = "local 2"
        print(b)

    g()
    print(a)
    print(b)
</code></pre>
