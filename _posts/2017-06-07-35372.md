<p>ものすごく簡単なことだと思いますが、問題に引っかかってしまっています。</p>

<p>現在python(version 3.6.0)を使い、アレイを作成しております。<br>
テキストファイルから読み込んだ値を、そのままアレイに挿入したいだけのことです。</p>

<p>具体的には、<br>
sample1.txtとsample2.txtがあるとします。</p>

```
sample1.txtには
20035.87109375,23184.52539062
20031.375,23162.98046875
```

<p>が値として入っており、</p>

```
sample2.txtには、
20037.72460938,23142.20117188
20051.09765625,23124.33203125
```

<p>が値として入っていると考えてください。</p>

<p>このファイルを読み込み、最終的には、以下のようなformatでアレイを作成したいです。</p>

```
[
 [[ 20035.87109375  23184.52539062]
  [ 20031.375       23162.98046875]]

 [[ 20037.72460938  23142.20117188]
  [ 20051.09765625  23124.33203125]]
]
```

<p>テキストファイルの値をアレイに挿入することはわかっています。</p>

```
もし数字で挿入しますと、
[20035.87109375, 23184.52539062]
```

<p>のような形になり、</p>

```
もし文字列で挿入しますと、
['20035.87109375 23184.52539062']
```

<p>のような形になってしまいます。<br>
問題は「数字」でも「文字列」でもないformatで挿入したいです（上記の例を参考にしてください）。</p>
