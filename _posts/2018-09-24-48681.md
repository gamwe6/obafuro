<p>Pythonでrandom関数を使ったじゃんけんプログラムを作っています。<br>
その中で今までの勝ち負け引き分けのそれぞれ合計を出す部分が</p>

<pre><code>Index Error : Taple index out of range 
</code></pre>

<p>というエラーを出していてその解決法が分かりません。</p>

<p>どなたかご指導お願いします。</p>

<pre><code># じゃんけんゲーム
import random

hand = ["グー", "チョキ", "パー", "ゲーム終了"]

win = 0
lose = 0
draw = 0
time = 0

print("===じゃんけんしましょう===")

while True:

    com = random.randint(0,2)
    for i, desc in enumerate(hand):
        print(i, ":", desc)
    you  = int(input("出す手を数値で入力: "))

    time += 1                       # 試行回数

    if you == 3: break
    if you &lt; 0 or you &gt; 2:
        print("0から3の間で入力してね")
        continue

    print("---")
    print("自分 : ", hand[you])
    print("相手 : ", hand[com])
    input("---")

    j = (you - com + 3) % 3
    if j == 0:
        print("あいこ")
        draw += 1               # 引き分けた回数
    elif j == 1:
        print("負け...")
        lose += 1               # 負けた回数
    elif j == 2:
        print("勝ち!!!")
        win += 1                # 勝った回数
    input("---")

    rate = win / time
    print("現在の勝率 : ", rate)     # 勝率
    s = win, "勝", lose, "敗", draw, "分"       #内訳。ここはエラーなし
    print(s)
    s2 = "{1}勝ち{2}負け{3}分け".format(win, lose, draw) # Tuple index out of range と出る
    print(s2)
    input("---")
</code></pre>
